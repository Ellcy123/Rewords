# 《词语炼金》玩法概念策划文档 V0.2

## 1. 项目定位

《词语炼金》是一款以"语义拆解、卡牌合成、剩余词处理"为核心的轻量化闯关策略游戏。

玩家在每个关卡节点中，从系统陈列的图文卡牌中选择有限资源。每张卡牌由策划预设，包含卡面图像、卡牌名称、可拆词语与合成规则。玩家通过拆解卡牌获得词语素材，再将词语重新组合成新的卡牌，用最终持有的卡牌组合触发当前节点的剧情分支。

本项目的核心体验可以概括为：

**拆卡成词，组词成卡，用卡改写剧情，用剩词承担代价。**

## 2. 参考产品与借鉴点

### 《Storyteller》

参考其"少量元素组合产生剧情结果"的直观表达方式，以及适合短视频传播的轻量交互结构。

### 《Baba Is You》

参考其"语言改变规则与结果"的设计启发，但本项目不采用复杂推箱子语法，而是转化为更易上手的语义卡牌合成。

### 《文字游戏》

参考其"中文文字本身具有可交互性"的表达潜力，但本项目不做全文字世界冒险，而是以图文卡牌、词语拆解和剧情分支为核心。

### 组词类 AI 评分玩法

参考其"离谱组合 + AI 反馈"的娱乐性，但本项目不让 AI 主导主线判定，而是将 AI 用于剩余词语处理与趣味反馈。

### 节点式卡牌闯关游戏

参考其"连续节点推进、资源选择、构筑取舍、长期后果"的结构优势，但本项目的核心不是战斗数值，而是词语资源的选择、合成与代价管理。

## 3. 核心玩法概述

每个关卡是一个独立剧情节点。

节点中会陈列若干张策划预设的图文卡牌，例如：

- 漂亮的裙子
- 破旧的裙子
- 项链
- 戒指
- 扫帚
- 邀请函
- 心动

玩家不能全部获得，只能根据关卡限制选择部分卡牌。

每张卡牌都包含：

- 卡牌名称
- 卡面图像
- 卡牌类型
- 可拆词语
- 可参与的合成规则
- 可触发的剧情标签

例如：

**漂亮的裙子**
- 图像：华丽礼裙
- 卡名：漂亮的裙子
- 可拆词：漂亮 / 裙子
- 类型：服装 / 体面
- 可触发：舞会入场、王子注意、贵族路线

**心动**
- 图像：跳动的心或脸红表情
- 卡名：心动
- 可拆词：心动
- 类型：状态 / 情感
- 可触发：王子一见钟情、情感分支

## 4. 核心循环

### 4.1 单节点循环

1. 进入剧情节点。
2. 查看当前节点目标。
3. 从陈列卡牌中选择有限数量的卡牌。
4. 拆解所选卡牌，获得词语素材。
5. 使用词语素材合成新卡牌。
6. 根据最终持有的卡牌组合触发节点分支。
7. 处理合成后剩余的词语。
8. 根据剩余词处理结果结算数值变化或失败惩罚。
9. 获得奖励、保留部分资源或进入下一节点。

### 4.2 长线循环

游戏采用连续节点推进结构。

每个节点的选择会影响：

- 当前剧情分支
- 后续可用卡牌
- 角色数值
- 路线倾向
- 特殊事件
- 最终结局

玩家需要在"当前节点最优解"和"后续资源保留"之间做取舍。

## 5. 卡牌系统

### 5.1 卡牌定义

卡牌是游戏中的核心资源单位。

每张卡牌都由策划预设，不是 AI 随机生成。

卡牌包含：

- 图像表现
- 卡牌名称
- 可拆词语
- 卡牌标签
- 可合成对象
- 可触发剧情条件

卡牌可以是具体物品，也可以是抽象概念。

### 5.2 卡牌类型

卡牌类型包括但不限于：

- **物品卡**：裙子、项链、戒指、扫帚、邀请函
- **属性卡**：漂亮、破旧、正式、脏、锋利
- **身份卡**：女仆、贵族、王子、骗子
- **状态卡**：心动、怀疑、害怕、愤怒
- **行为卡**：看、送、偷、藏、骗
- **关系卡**：喜欢、信任、误会、背叛

其中部分卡牌可以直接使用，部分卡牌可以被拆解，部分卡牌只能作为剧情触发条件存在。

## 6. 拆词与合成规则

### 6.1 拆词

玩家获得卡牌后，可以将部分卡牌拆解为词语素材。

拆解存在两个层级：

**词语级拆解：** 将卡牌拆为完整词语。

- 漂亮的裙子 → 漂亮 / 裙子
- 破旧的裙子 → 破旧 / 裙子
- 正式的邀请函 → 正式 / 邀请函

**字级拆解（主要模式）：** 将词语拆为单字素材，是大部分合成的基础。

- 裙子 → 裙 / 子
- 女仆 → 女 / 仆
- 心动 → 心 / 动
- 背叛 → 背 / 叛

字级拆解后，单字可以跨词重新组合，产生与原词完全不同的新含义。这是本作核心玩法的基础操作。

拆解会破坏原卡牌，并将其转化为可用于合成的新素材。策划可根据关卡需要控制每张卡牌允许的拆解层级。

### 6.2 合成

玩家可以将词语素材重新组合，生成新的卡牌。

合成的核心模式是**拆字重组**——从不同词语中取出单字，组成全新含义的词。大部分关卡的关键合成都基于这一模式，而非简单的修饰词替换。

**拆字重组（核心模式）：** 从不同词语中取出单字，组成全新含义的词。

- 裙**子** + **女**仆 → **女子**（物品+身份 → 全新身份）
- **心**动 + 怀**疑** → **疑心**（情感变为猜忌）
- **背**叛 + **心**动 → **背心**（抽象概念变为具体物品）
- **戒**指 + **心**动 → **戒心**（饰品+情感 → 警觉防备）
- **怀**疑 + 破**旧** → **怀旧**（猜忌变为感伤）

同一组素材因取字方式不同，可以走向完全不同的分支。这是本作区别于普通合成游戏的核心设计——**玩家不是在选配方，而是在字的碎片里寻找新的意义。**

**动作合成：** 行为词 + 物品/身份，产生事件性质的卡牌。

- 偷 + 戒指 → 偷来的戒指
- 藏 + 邀请函 → 藏起来的邀请函
- 骗 + 贵族 → 骗过贵族

**关系合成：** 关系词 + 身份/状态，改变人物关系走向。

- 信任 + 骗子 → 信任骗子
- 误会 + 王子 → 误会王子
- 喜欢 + 女仆 → 喜欢女仆

**修饰合成（辅助模式）：** 属性词 + 名词，改变物品性质。部分关卡中作为基础合成存在，但不是主要策略点。

- 漂亮 + 裙子 → 漂亮的裙子
- 破旧 + 邀请函 → 破旧的邀请函

核心合成结果应由策划预设，保证关卡可控。

AI 不负责生成主线关键卡牌，也不负责决定核心合成是否成立。

### 6.3 合成结果

合成成功后，玩家获得一张新卡牌。

新卡牌会参与当前节点的剧情分支判定。

同样的原始卡牌，因拆解和合成方式不同，可以走向完全不同的结果：

**示例：玩家选择了"裙子"和"女仆"两张卡**

词语级合成：
- 女仆 + 裙子 → 女仆的裙子 → 触发：被当作工作人员，从后门进入舞会。

字级重组：
- 裙**子** + **女**仆 → **女子** → 触发：以神秘女子身份出现，引起贵族注意。

**同样的卡牌，拆法决定命运。** 这是"怎么拆"比"选什么"更重要的核心策略点。

更多合成触发示例：

- 漂亮的裙子 + 心动 → 王子对玩家一见钟情。
- 偷来的戒指 + 漂亮的裙子 → 玩家伪装贵族入场，但埋下被揭穿的隐患。
- 信任骗子 → 被骗子利用，进入骗局支线。
- 疑心 + 王子 → 王子开始调查玩家身份。
- 锋利的扫帚 → 开启非常规的武器路线或荒诞分支。

## 7. 剩余词语处理

### 7.1 规则目的

玩家拆解卡牌并合成新卡后，未被使用的词语不能直接消失。

这些剩余词语需要被玩家处理。

该系统用于制造代价感，避免玩家只拿走有利词语，而不承担拆解后的后果。

核心原则是：

**合成目标卡牌是收益，处理剩余词语是代价。**

### 7.2 基础组句词

在剩余词语处理阶段，系统默认提供 4 个基础词：

**我 / 你 / 在 / 了**

这些基础词不消耗，不参与主要构筑，只用于帮助玩家处理剩余词语。

例如：

玩家使用：
- 漂亮的项链
- 破旧的裙子

拆解后获得：
- 漂亮
- 项链
- 破旧
- 裙子

玩家合成：
- 漂亮 + 裙子 → 漂亮的裙子

剩余词语为：
- 破旧
- 项链

玩家需要使用基础词处理剩余词语，例如：
- 破旧项链在我了
- 你在破旧项链了
- 我在破旧项链了

这些句子不一定自然，但需要由 AI 判断其合理程度、荒诞程度和负面影响。

### 7.3 AI 评分触发点

AI 评分主要发生在"剩余词语处理"阶段，而不是主卡牌合成阶段。

评分维度包括：

- **合理分**：句子是否勉强说得通。
- **抽象分**：句子是否荒诞、离谱或有节目效果。
- **诗意分**：句子是否有意象或特殊表达感。
- **负担分**：该表达是否会造成尴尬、怀疑或负面后果。

如果综合结果过低，系统可能触发惩罚。

惩罚包括：

- 体面下降
- 怀疑上升
- 压力上升
- 当前分支变差
- 触发负面事件
- 当前节点失败

## 8. 节点结构

本项目采用"单节点多分支"的剧情结构。

每个节点只呈现一个当前事件，不做多个地点并行分配。

玩家在当前节点中通过卡牌选择、拆词合成和剩余词处理，决定该节点的分支结果。

### 示例节点：舞会入口

**节点目标：** 设法进入舞会。

**陈列卡牌：**
- 漂亮的裙子
- 破旧的裙子
- 项链
- 戒指
- 扫帚
- 邀请函
- 心动

玩家根据所选卡牌和合成结果可能触发不同分支：

**分支 A：贵族路线**
- 持有：漂亮的裙子、邀请函
- 结果：门卫认为玩家身份体面，允许从正门入场。
- 体面上升。

**分支 B：女仆路线**
- 持有：女仆的裙子、扫帚
- 结果：玩家被当作工作人员，从后门进入。
- 体面不上升，但怀疑降低。

**分支 C：恋爱路线**
- 持有：漂亮的裙子、心动
- 结果：王子注意到玩家，并触发一见钟情剧情。
- 开启王子相关分支。

**分支 D：尴尬路线**
- 持有：扫帚
- 结果：门卫嘲笑玩家是来打扫卫生的。
- 体面下降，但可能开启仆人支线。

**分支 E：异常路线**
- 持有：扫帚、心动
- 结果：根据上下文可触发荒诞分支，例如王子对玩家的怪异气质产生兴趣，或对扫帚产生异常关注。
- 该类分支可根据项目调性决定是否保留。

## 9. 数值系统

数值系统应保持轻量，用于承接节点之间的长期影响。

建议初期只保留 2–4 个核心数值。

可选数值包括：

- **体面**
- **怀疑**
- **压力**
- **灵感**
- **金钱**

数值用途：

- 影响节点分支门槛
- 影响后续卡牌选择
- 影响结局倾向
- 承接剩余词语处理失败的惩罚

例如：
- 体面过低，无法走贵族路线。
- 怀疑过高，会被门卫或贵族盘问。
- 压力过高，节点失败概率上升。
- 灵感较高，可以允许一次特殊合成或异常分支。

## 10. AI 使用边界

AI 是辅助反馈系统，不是核心规则系统。

**AI 负责：**
- 判断剩余词语组成的句子是否合理。
- 给出合理分、抽象分、诗意分、负担分。
- 生成短反馈文本。
- 处理非关键性的荒诞结果。
- 增强传播效果。

**AI 不负责：**
- 随机生成主线卡牌。
- 决定关键剧情分支。
- 决定核心合成结果。
- 替代策划配置关卡规则。
- 完全决定玩家是否通关。

核心玩法必须由策划规则保证可控，AI 只承担弹性反馈与内容增强。

## 11. 玩家决策点

玩家在每个节点中主要面对以下决策：

1. 从陈列卡牌中选择哪些卡牌。
2. 哪些卡牌直接保留，哪些卡牌拆解。
3. 使用哪些词语合成新卡牌。
4. 是否为了当前节点消耗关键词语。
5. 如何处理合成后的剩余词语。
6. 选择稳定路线、低风险路线、情感路线或异常路线。
7. 在当前收益与后续风险之间取舍。

游戏策略性来自：

**有限卡牌选择 + 词语合成 + 剩余词代价处理 + 节点分支反馈。**

## 12. 原型设计建议

第一版原型目标是验证核心玩法是否成立。

建议规模：

- 节点数量：3–5 个
- 每个节点陈列卡牌：5–7 张
- 每个节点可选卡牌：2–3 张
- 每个节点主要分支：3–5 个
- 可拆词语数量：20–30 个以内
- 核心数值：2–3 个
- AI 评分只用于剩余词语处理

第一版原型重点验证：

1. 玩家是否理解卡牌拆词与合成。
2. 玩家是否能理解剩余词处理的代价。
3. 单节点多分支是否有趣。
4. AI 评分是否能制造传播点。
5. 卡牌选择是否产生真实取舍。
6. 该玩法是否具备继续扩展关卡的潜力。

## 13. 设计风险

### 13.1 玩法变成普通合成游戏

如果只做"两个词合成一个新词"，玩法会过轻。

**解决方式：** 加入剩余词语处理，让每次合成都产生代价。

### 13.2 AI 判定不稳定

AI 可能对同一句话给出不同判断。

**解决方式：** AI 只处理剩余词反馈，不参与主线硬判定。

### 13.3 卡牌组合过于自由

组合过多会导致关卡失控。

**解决方式：** 关键合成结果由策划预设，自由组合只进入低权重反馈。

### 13.4 分支制作量膨胀

每个节点多分支可能增加剧情文本压力。

**解决方式：** 每个节点只保留 3–5 个主分支，其余组合通过短反馈、数值变化或失败处理。

### 13.5 玩家不理解剩余词处理

剩余词处理规则比较新，需要清晰教学。

**解决方式：** 第一关只做一次非常直观的示例，例如"漂亮的项链 + 破旧的裙子 → 漂亮的裙子，剩下破旧的项链需要处理"。

## 14. 当前版本玩法定义

《词语炼金》是一款语义卡牌合成闯关游戏。

玩家在每个剧情节点中，从策划预设的图文卡牌中选择有限资源。卡牌可以被拆解为词语素材，词语素材可以重新合成为新的卡牌。当前节点会根据玩家最终持有的卡牌组合触发不同剧情分支。合成后剩余的词语需要使用极少基础词进行处理，并由 AI 评分决定代价、惩罚或失败结果。

核心体验可以概括为：

**用有限的词炼成关键卡牌，再用剩下的词承担代价。**
