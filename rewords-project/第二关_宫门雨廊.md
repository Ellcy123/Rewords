# 第二关：宫门雨廊

## 关卡定位

**故事前提：** 女孩终于来到贵族舞会门前。宫门外下着细雨，车轮碾过水洼，侍从念着名单，贵族小姐们在雨廊下等待入场。她已经有了第一关炼出的某种身份或物件，但真正的舞会还隔着一道门。

**节点目标：** 在门卫盘问前，选择一种入场方式。

**设计重点：**
- 第二关比第一关略复杂，但仍控制字池，避免一下子变成大字堆。
- 分支必须围绕“宫门雨廊”展开：门、名单、后廊、侍从、女伴、舞曲。
- 每个分支都有明确指向：官方入场、绕路入场、工作人员入场、社交入场。
- 每个分支提供一段不超过 1 分钟的演出文本，用于承接玩家选择。

**核心数值：**
- 体面：影响她能站在舞会的哪个圈层。
- 怀疑：影响门卫和贵族是否追问她的身份。
- 灵感：影响她能否把尴尬变成童话事件。

---

## 关卡一到关卡二承接演出

根据第一关结果，可在进入本关前播放一段短演出。若第一关合成了 2 张结果卡，优先播放更高戏剧性的承接文本，或混合两条文本中的一句。

### 承接 A：灰姑娘

**演出文本：**

雨落在宫门前的石阶上。女孩低头看见裙角还有一点灰，她伸手想拍掉，却听见身后有人小声说：“那是谁家的姑娘？”

旁白：灰没有离开她。可今晚，它不再只像贫穷，也像一个故事的开头。

### 承接 B：水晶鞋

**演出文本：**

雨水映着她鞋尖的一点亮光。两个贵族小姐同时回头，其中一个低声说：“那双鞋……是玻璃做的吗？”

女孩把脚往裙下藏了藏。鞋光却比她更诚实，先一步走进了众人的目光里。

### 承接 C：请柬

**演出文本：**

门卫伸手：“请柬。”

女孩把那张修补过的纸递出去。纸边还不平整，烫金却在雨里亮了一下。门卫没有立刻放行，也没有立刻赶她走。

旁白：规则认得这张纸。现在只差一个让人相信的名字。

### 承接 D：贵族姑娘

**演出文本：**

她站进贵族小姐们的队伍里，背挺得很直。雨廊下有人打量她的袖口，又打量她的鞋。

“您是哪一家的小姐？”一个声音轻轻问。

女孩微笑了一下。这个笑很贵，但还没有姓氏。

## 一、分支剧情

### 分支 A：名单入场

**触发结果卡：** 名单

**结果状态：** 女孩被写进门口的临时名单，获得官方认可的入场理由。

**明确指向：** 官方入场 / 低风险 / 规则路线

**后续影响：**
- 进入舞会正厅外围。
- 后续更容易接触管家、名册、请柬、座次表等秩序资源。
- 若第一关持有“请柬”，本分支怀疑额外 -1。

**数值变化：**
- 体面 +1
- 怀疑 -2

**分支演出文本：**

门卫翻过名册，又看了一眼她手里的纸。

“名字？”

女孩没有立刻回答。雨声替她停了一拍。随后，名册上的空行像被墨水轻轻推开，露出一个刚好能容纳她的位置。

门卫皱眉，却还是侧身让开：“进去吧。别在门口停太久。”

旁白：她不是被欢迎的，但她被记录了。对宫门来说，这已经足够。

### 分支 B：后门入场

**触发结果卡：** 后门

**结果状态：** 女孩没有从正门接受审视，而是从雨廊尽头的后门进入舞会边缘。

**明确指向：** 绕路入场 / 低体面 / 低关注

**后续影响：**
- 进入仆役走廊或厨房侧厅。
- 后续更容易获得厨房、侍从、杂物间相关场景卡。
- 后续接近王子较难，但怀疑增长较慢。

**数值变化：**
- 体面 -1
- 怀疑 -1
- 灵感 +1

**分支演出文本：**

雨廊尽头有一扇小门，门缝里漏出暖黄的光和烤面包的气味。

一个侍从抱着银盘匆匆经过，没有看她，只低声说：“别挡路。”

女孩顺着那句话往前走。没有人宣布她的到来，也没有人阻止她的脚步。

旁白：正门留给贵族。后门留给忙碌的人，也留给不想被看见的奇迹。

### 分支 C：侍女入场

**触发结果卡：** 侍女

**结果状态：** 女孩被临时认作舞会侍女，可以进入会场服务区，但身份被压低。

**明确指向：** 工作人员身份 / 资源获取 / 体面牺牲

**后续影响：**
- 进入侍女和侍从的工作路线。
- 后续更容易听到贵族秘密、获得酒杯、托盘、钥匙等资源。
- 王子初见不会立刻发生，但可开启“被忽视者偷听”玩法。

**数值变化：**
- 体面 -2
- 怀疑 -2
- 灵感 +1

**分支演出文本：**

侍从铃响了一声。有人把一条披肩塞进她怀里：“你，去雨廊那边等着，小姐们进门后把湿披肩收起来。”

女孩张了张口，最后没有解释。

她站到灯影边缘，看着贵族们一个个从她面前经过。没人问她是谁，因为没人觉得侍女需要有名字。

旁白：这不是体面的入口。但被忽视的人，常常能看见最多秘密。

### 分支 D：舞伴入场

**触发结果卡：** 舞伴

**结果状态：** 女孩借由“舞伴”身份被带入门内。她不必证明家族，只要有人愿意与她同入。

**明确指向：** 社交入场 / 高机会 / 高不确定

**后续影响：**
- 进入舞池外围，王子或陌生贵族更容易注意到她。
- 若第一关持有“水晶鞋”，本分支体面额外 +1。
- 后续可能触发“第一支舞”或“被认错的舞伴”事件。

**数值变化：**
- 体面 +2
- 怀疑 +1
- 灵感 +1

**分支演出文本：**

雨廊里传来舞曲的前奏。一个年轻贵族回头，看见她站在门边，像是迟到，又像是在等谁。

“小姐，您也是一个人？”

他伸出手。女孩看着那只手，又看向门卫。门卫没有再问请柬，只把目光移向下一辆马车。

旁白：有些门需要名字打开。有些门，只需要一支舞的开始。

## 二、结果卡牌

### 1. 名单

**卡面图像：** 雨滴落在门口名册上，空白的一行浮出淡淡墨迹。

**合成配方：**
- 名（来源：名册）
- 单（来源：礼单）

**对应分支：** 名单入场

**数值效果：**
- 体面 +1
- 怀疑 -2

**啊哈点：** “名册”给出“名”，“礼单”给出“单”，两个门口文书拼成真正能放行的“名单”。

### 2. 后门

**卡面图像：** 宫门的正面仍然拥挤，雨廊尽头却亮起一扇窄窄的小门。

**合成配方：**
- 后（来源：后廊）
- 门（来源：宫门）

**对应分支：** 后门入场

**数值效果：**
- 体面 -1
- 怀疑 -1
- 灵感 +1

**啊哈点：** “宫门”太正式，“后廊”太不起眼；拼成“后门”后，玩家马上理解这是绕开盘问的路。

### 3. 侍女

**卡面图像：** 侍从铃旁，一条女伴披肩被递到她手上，她的身份被临时改写成服务者。

**合成配方：**
- 侍（来源：侍从铃）
- 女（来源：女伴披肩）

**对应分支：** 侍女入场

**数值效果：**
- 体面 -2
- 怀疑 -2
- 灵感 +1

**啊哈点：** “侍从”里取“侍”，“女伴”里取“女”，拼成“侍女”。这是身份降级，但也是低风险潜入。

### 4. 舞伴

**卡面图像：** 门外舞曲响起，披肩的主人迟迟未到，女孩获得了一个空出来的入场位置。

**合成配方：**
- 舞（来源：门外舞曲）
- 伴（来源：女伴披肩）

**对应分支：** 舞伴入场

**数值效果：**
- 体面 +2
- 怀疑 +1
- 灵感 +1

**啊哈点：** “门外舞曲”提供“舞”，“女伴披肩”提供“伴”，拼成“舞伴”，从物件和声音里炼出一个社交身份。

## 三、双结果组合分支

玩家最多可合成 2 张结果卡。若合成 2 张，优先触发组合分支。

### 名单 + 舞伴：体面入场

**结果状态：** 她既在名单上，又有人愿意与她同入。门卫不再盘问，贵族们也默认她有资格出现。

**追加影响：**
- 体面额外 +1。
- 后续进入舞池外围。
- 若怀疑低于 0，可触发“第一支舞”节点。

### 名单 + 后门：有记录的绕行

**结果状态：** 名义上她被允许入场，实际上却从后门避开了人群。

**追加影响：**
- 怀疑额外 -1。
- 后续进入侧厅，可以选择是否转入正厅。
- 失去一次立刻吸引王子注意的机会。

### 后门 + 侍女：仆役路线

**结果状态：** 她从后门进入，并被当作侍女安排工作。没人关注她，但她能接近很多贵族听不到的地方。

**追加影响：**
- 开启“偷听秘密”标签。
- 后续更容易获得钥匙、托盘、酒杯类场景物品。
- 体面额外 -1。

### 侍女 + 舞伴：被认错的人

**结果状态：** 她一边像侍女，一边又被误认为某人的舞伴。身份出现矛盾，但也制造了机会。

**追加影响：**
- 开启“身份错位”标签。
- 灵感额外 +1。
- 后续怀疑事件更容易出现，但可触发荒诞反馈。

### 名单 + 侍女：名单上的侍女

**结果状态：** 她不是贵族名单上的客人，而是临时服务名单上的人。门卫放行，但不会给她客人的待遇。

**追加影响：**
- 怀疑额外 -1。
- 后续可在服务区自由移动。
- 无法直接进入舞池中心。

### 后门 + 舞伴：迟到的邀约

**结果状态：** 她从后门边缘进入，却在门内被人当作迟到的舞伴带走。

**追加影响：**
- 灵感额外 +1。
- 可从侧厅切入舞池。
- 后续“你从哪里进来的？”盘问权重提高。

## 六、资源分配感

本关的资源分配重点不是“字够不够”，而是“玩家愿意把入场资格投向哪一种路线”。

- **名单**：消耗文书资源，换低怀疑。适合稳进门，但戏剧性较低。
- **后门**：消耗空间资源，换低关注。适合进入侧厅和仆役路线。
- **侍女**：消耗身份资源，牺牲体面，换取情报和移动自由。
- **舞伴**：消耗社交资源，提高体面和机会，但带来怀疑。

选择 2 张结果卡时，玩家会更明显地决定自己的入场风格：
- `名单 + 舞伴`：最体面
- `后门 + 侍女`：最隐蔽
- `侍女 + 舞伴`：最戏剧化
- `名单 + 后门`：最安全但不够高光

## 七、剩余字处理示例

基础组句词：**我 / 你 / 在 / 了**

### 示例 1：合成“名单”

**消耗字：** 名 / 单

**剩余字：** 宫 / 门 / 后 / 廊 / 册 / 礼 / 侍 / 从 / 铃 / 女 / 伴 / 披 / 肩 / 门 / 外 / 舞 / 曲

**可尝试句：**
- 我在宫门后廊了。
- 侍从铃女伴披肩在门外舞曲。

**预期反馈：**
- 合理分中，诗意分中。
- 因剩余字较多，可能有轻微负担，但路线稳定。

### 示例 2：合成“后门 + 侍女”

**消耗字：** 后 / 门 / 侍 / 女

**剩余字：** 宫 / 廊 / 名 / 册 / 礼 / 单 / 从 / 铃 / 伴 / 披 / 肩 / 门 / 外 / 舞 / 曲

**可尝试句：**
- 我在宫廊了，名册礼单从铃。
- 伴披肩在门外舞曲。

**预期反馈：**
- 合理分低，抽象分中。
- 剩下“名册礼单”暗示她放弃了官方身份，后续更偏侧厅路线。

### 示例 3：合成“名单 + 舞伴”

**消耗字：** 名 / 单 / 舞 / 伴

**剩余字：** 宫 / 门 / 后 / 廊 / 册 / 礼 / 侍 / 从 / 铃 / 女 / 披 / 肩 / 门 / 外 / 曲

**可尝试句：**
- 我在宫门后廊了。
- 侍从铃女披肩在门外曲。

**预期反馈：**
- 合理分中低，负担较低。
- 这是较体面的路线，但“后廊”和“侍从铃”仍可能成为后续被追问的隐患。

## 八、设计自检

- [x] 从剧情分支开始设计。
- [x] 结果卡牌恰好 4 张。
- [x] 每张结果卡牌都指向独立剧情分支。
- [x] 每个分支都有明确路线指向：官方、绕路、工作人员、社交。
- [x] 每个分支有不超过 1 分钟的演出文本。
- [x] 所有分支剧情都符合“宫门雨廊 / 舞会入口”场景。
- [x] 结果卡像自然词或自然短语：名单、后门、侍女、舞伴。
- [x] 场景物品为 7 张，总字池 19 个，略高于第一关但仍清爽。
- [x] 存在资源分配感：文书、空间、身份、社交四种入场资源。
- [x] 存在多种 2 张结果卡组合，并配置组合分支。
- [x] 非配方表中的组合不能合成结果卡。
