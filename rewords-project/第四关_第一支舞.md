# 第四关：第一支舞

## 关卡定位

**故事前提：** 第一支舞即将开始。乐声压过谈话，银灯把舞池照得像一只打开的盒子。女孩已经在正厅边缘取得了某种立足点，但现在所有选择都会被看见：伸手、退后、摘下面具，或悄悄交换一只酒杯。

**节点目标：** 在第一支舞开始时，决定自己如何处理“被看见”这件事。

**设计重点：**
- 第四关承接第三关的舞会资本：注意力、掩护、情报、人情。
- 所有分支都发生在“第一支舞 / 舞池边缘 / 退席通道 / 酒杯托盘”范围内。
- 分支指向要清晰：高光共舞、安全退场、身份揭露、秘密交换。
- 每个分支都有不超过 1 分钟的演出文本。
- 本关不再扩大字池，而是让资源选择变得更尖锐。

**核心数值：**
- 体面：决定她是否能被当作舞会焦点。
- 怀疑：决定她是否会被追问身份。
- 灵感：决定她是否能把危机变成童话时刻。
- 人情：决定她能否借社交关系换取后续保护。

---

## 第三关到第四关承接演出

根据第三关结果，进入本关前播放一段短演出，并影响第四关的初始倾向。若第三关合成 2 张结果卡，优先播放组合结果中更强势的路线文本，或混合一句。

### 承接 A：王子

**演出文本：**

王子的目光穿过银灯和人群，停在她身上。乐师像是等到了某个信号，第一支舞的前奏忽然变得清晰。

旁白：被王子看见，是机会，也是灯光。灯光会照亮她，也会照亮她身后的灰。

**第四关倾向：**
- 选择“共舞”时体面额外 +1。
- 选择“退场”时会触发“王子追问”隐患。

### 承接 B：假面

**演出文本：**

假面遮住了她的表情，也遮住了她迟疑的一瞬。有人从她身边经过，没认出她，却忍不住回头看第二眼。

旁白：看不清的人，最容易被想象。可想象太久，也会变成盘问。

**第四关倾向：**
- 选择“揭面”时怀疑额外 +1，但体面额外 +1。
- 选择“退场”时怀疑额外 -1。

### 承接 C：密信

**演出文本：**

密信藏在她袖口里，纸角贴着手腕。舞池里每一次转身，都像有人快要碰到那个秘密。

旁白：秘密不重，却很烫。它让她站得更稳，也让她不敢站太久。

**第四关倾向：**
- 选择“换杯”时人情额外 +1。
- 选择“共舞”时怀疑额外 +1。

### 承接 D：礼物

**演出文本：**

小礼盒被她握在掌心，金线压出一道浅痕。有人看见了，以为她正准备把它送给谁。

旁白：礼物是一句还没说出口的话。今晚，说给谁听，会改变她站在哪一边。

**第四关倾向：**
- 选择“换杯”或“共舞”时体面额外 +1。
- 选择“退场”时失去一次人情机会。

## 一、分支剧情

### 分支 A：高光共舞

**触发结果卡：** 共舞

**结果状态：** 女孩与关键人物进入舞池共舞。她获得全场可见的体面，也把自己放进所有人的谈话里。

**明确指向：** 恋爱高光 / 舞池中心 / 高收益高风险

**后续影响：**
- 开启“第一支舞”路线。
- 若第三关持有“王子”，本分支可直接触发王子对话节点。
- 贵族小姐阵营敌意上升，后续更容易被针对。

**数值变化：**
- 体面 +3
- 怀疑 +2
- 灵感 +1

**分支演出文本：**

他向她伸出手。周围的谈话像被银灯压低，只剩下乐声一步一步靠近。

女孩把手放上去。

舞池让开一小圈。她跟着第一拍旋转，裙角扫过灯影，刚才还怀疑她的人忽然安静下来。

旁白：这一刻，她不需要解释自己是谁。会跳舞的人，暂时只需要跟上节拍。

### 分支 B：安全退场

**触发结果卡：** 退场

**结果状态：** 女孩主动离开舞池焦点，避开即将到来的盘问。她失去高光机会，但保住当前身份。

**明确指向：** 安全 / 延后冲突 / 侧厅路线

**后续影响：**
- 进入侧厅或退席通道。
- 后续更容易接触仆役、失物、密信、厨房资源。
- 若第三关持有“假面”，本分支怀疑额外 -1。

**数值变化：**
- 体面 -1
- 怀疑 -2
- 灵感 +1

**分支演出文本：**

乐声刚起，她却往后退了一步。

贵族们以为她只是让出舞池，没人立刻阻拦。退席通道的帘子被风掀开，露出一条没有灯光的路。

她听见身后有人问：“那位小姐怎么走了？”

旁白：有时，不被记住也是一种胜利。至少在钟声敲响之前。

### 分支 C：主动揭面

**触发结果卡：** 揭面

**结果状态：** 女孩选择摘下面具或揭开某层伪装。她用主动暴露换取短暂信任，但身份问题会提前到来。

**明确指向：** 身份真相 / 信任交换 / 中高风险

**后续影响：**
- 开启“身份问答”节点。
- 若前置持有“灰姑娘”或“假面”，本分支灵感额外 +1。
- 后续不可再使用完全匿名的路线。

**数值变化：**
- 体面 +1
- 怀疑 +3
- 人情 +1

**分支演出文本：**

她伸手碰到假面边缘。

有人轻轻吸了一口气。有人停下扇子。银灯把她的脸照出来，也把她袖口的旧痕、裙边的细灰，一并照出来。

她没有低头。

旁白：谎言可以保护她一会儿。可主动揭开的真相，有时也能让别人欠她一次沉默。

### 分支 D：暗中换杯

**触发结果卡：** 换杯

**结果状态：** 女孩在舞曲和人群掩护下完成一次酒杯交换。她获得秘密、暗号或人情，但行为本身带来隐患。

**明确指向：** 秘密交易 / 人情资源 / 暗线推进

**后续影响：**
- 开启“酒杯暗号”或“侧厅交易”节点。
- 若第三关持有“密信”或“礼物”，本分支人情额外 +1。
- 后续可能被管家阵营追查。

**数值变化：**
- 人情 +2
- 灵感 +1
- 怀疑 +1

**分支演出文本：**

托盘从她身边经过。银杯在灯下排成一列，每只都像没有名字。

女孩伸手，把其中一杯轻轻换到另一只位置。

动作很小，小到只有端着托盘的侍从看见。他没有出声，只把目光落在她袖口那封信的位置。

旁白：舞曲盖住脚步，也盖住交易。只要杯子还没被举起，一切都可以假装没有发生。

## 二、结果卡牌

### 1. 共舞

**卡面图像：** 琴弦的共鸣推开舞池灯影，两个人的影子在银灯下重叠。

**合成配方：**
- 共（来源：共鸣琴弦）
- 舞（来源：银灯舞池）

**对应分支：** 高光共舞

**数值效果：**
- 体面 +3
- 怀疑 +2
- 灵感 +1

**啊哈点：** “共鸣琴弦”的“共”和“银灯舞池”的“舞”拼成“共舞”，从音乐和场地里炼出高光行动。

### 2. 退场

**卡面图像：** 退席牌旁的会场地毯延伸到帘幕后方，像给她留出一条暗路。

**合成配方：**
- 退（来源：退席牌）
- 场（来源：会场地毯）

**对应分支：** 安全退场

**数值效果：**
- 体面 -1
- 怀疑 -2
- 灵感 +1

**啊哈点：** “退席牌”给出退，“会场地毯”给出场，拼成“退场”，是玩家一眼能明白的安全撤离选择。

### 3. 揭面

**卡面图像：** 揭幕绳一拉，银面具后的脸被灯光照亮。

**合成配方：**
- 揭（来源：揭幕绳）
- 面（来源：银面具）

**对应分支：** 主动揭面

**数值效果：**
- 体面 +1
- 怀疑 +3
- 人情 +1

**啊哈点：** “揭幕”和“面具”拼出“揭面”，从舞会布景和伪装道具里炼出主动暴露身份的动作。

### 4. 换杯

**卡面图像：** 托盘上的两只银杯交换位置，其中一只杯沿留下淡淡口红印。

**合成配方：**
- 换（来源：换杯托盘）
- 杯（来源：换杯托盘）

**对应分支：** 暗中换杯

**数值效果：**
- 人情 +2
- 灵感 +1
- 怀疑 +1

**啊哈点：** “换杯托盘”本身就藏着一个动作结果，作为低门槛合成，让玩家直观理解“物品名里能提炼行动”。

## 三、双结果组合分支

玩家最多可合成 2 张结果卡。若合成 2 张，优先触发组合分支。

### 共舞 + 揭面：灯下真相

**结果状态：** 她在全场目光中共舞，并主动揭开伪装。高光变成公开宣告，危险也提前到来。

**追加影响：**
- 体面额外 +1。
- 后续强制进入“身份问答”节点。
- 王子若在场，会优先替她发问或替她解围。

### 共舞 + 退场：一舞即逃

**结果状态：** 她完成第一支舞后立刻退场，留下足够强的印象，也留下一个空白。

**追加影响：**
- 开启“寻找舞伴”标签。
- 怀疑本次不再额外增加。
- 后续可触发“遗失线索”事件。

### 共舞 + 换杯：舞步掩护

**结果状态：** 她借共舞的旋转完成杯子交换。众人只看见舞步，没有看见交易。

**追加影响：**
- 人情额外 +1。
- 若第三关持有“密信”，开启“暗号传递”标签。
- 后续管家阵营警觉上升。

### 退场 + 揭面：留名离开

**结果状态：** 她在退场前主动露出真容，让别人记住她，却不给他们当场追问的机会。

**追加影响：**
- 体面 +1。
- 怀疑延后到下一节点结算。
- 后续会出现“追来的提问者”。

### 退场 + 换杯：带走证据

**结果状态：** 她避开舞池焦点，同时完成杯子交换，把真正有用的线索带离正厅。

**追加影响：**
- 怀疑额外 -1。
- 人情 +1。
- 后续进入侧厅调查路线。

### 揭面 + 换杯：当面交易

**结果状态：** 她不再隐藏自己，同时完成一次明确交换。对方知道她是谁，也知道她手里有筹码。

**追加影响：**
- 开启“明牌交易”标签。
- 人情额外 +1。
- 怀疑额外 +1。

## 六、资源分配感

本关的资源分配重点是：玩家如何处理已经到来的“可见性”。

- **共舞**：把可见性变成体面和机会，但会提高怀疑。
- **退场**：牺牲高光，保住安全和后续空间。
- **揭面**：用真相换信任或人情，但身份问题提前爆发。
- **换杯**：把舞会动作转为暗线交易，收益偏人情和线索。

二卡组合让玩家决定自己是如何“被记住”的：
- `共舞 + 揭面`：公开高光
- `共舞 + 退场`：留下传闻
- `退场 + 换杯`：带走线索
- `揭面 + 换杯`：明牌交易

## 七、剩余字处理示例

基础组句词：**我 / 你 / 在 / 了**

### 示例 1：合成“共舞”

**消耗字：** 共 / 舞

**剩余字：** 银 / 灯 / 池 / 鸣 / 琴 / 弦 / 退 / 席 / 牌 / 会 / 场 / 地 / 毯 / 银 / 面 / 具 / 揭 / 幕 / 绳 / 换 / 杯 / 托 / 盘

**可尝试句：**
- 我在银灯池鸣琴弦了。
- 退席牌会场地毯在银面具揭幕绳换杯托盘。

**预期反馈：**
- 合理分低，诗意分中。
- 只抓高光会留下大量舞会结构和退路资源，负担较高。

### 示例 2：合成“共舞 + 退场”

**消耗字：** 共 / 舞 / 退 / 场

**剩余字：** 银 / 灯 / 池 / 鸣 / 琴 / 弦 / 席 / 牌 / 会 / 地 / 毯 / 银 / 面 / 具 / 揭 / 幕 / 绳 / 换 / 杯 / 托 / 盘

**可尝试句：**
- 银灯池在我了，鸣琴弦席牌会地毯。
- 银面具揭幕绳换杯托盘在你。

**预期反馈：**
- 合理分中低，负担中。
- 这条路线有戏剧性：她留下舞的印象，也留下退场后的空白。

### 示例 3：合成“揭面 + 换杯”

**消耗字：** 揭 / 面 / 换 / 杯

**剩余字：** 银 / 灯 / 舞 / 池 / 共 / 鸣 / 琴 / 弦 / 退 / 席 / 牌 / 会 / 场 / 地 / 毯 / 银 / 具 / 幕 / 绳 / 托 / 盘

**可尝试句：**
- 我在银灯舞池共鸣琴弦。
- 退席牌会场地毯银具幕绳托盘在你了。

**预期反馈：**
- 抽象分高，怀疑负担高。
- 适合明牌交易路线，但会放弃“共舞”和“退场”的主动节奏。

## 八、设计自检

- [x] 从剧情分支开始设计。
- [x] 结果卡牌恰好 4 张。
- [x] 每张结果卡牌都指向独立剧情分支。
- [x] 每个分支都有明确路线指向：高光共舞、安全退场、主动揭面、暗中换杯。
- [x] 每个分支有不超过 1 分钟的演出文本。
- [x] 所有分支剧情都符合“第一支舞 / 舞池边缘”场景。
- [x] 结果卡像自然词或自然短语：共舞、退场、揭面、换杯。
- [x] 场景物品为 7 张，总字池 25 个，复杂度接近第三关但没有继续膨胀。
- [x] 存在资源分配感：高光、安全、真相、交易四种处理方式。
- [x] 存在多种 2 张结果卡组合，并配置组合分支。
- [x] 非配方表中的组合不能合成结果卡。
