# 第三关：银灯舞池

## 关卡定位

**故事前提：** 女孩进入舞会正厅边缘。银灯照亮舞池，乐师调弦，贵族们在笑声里交换眼神。她已经通过宫门，但“进来了”不等于“站稳了”。现在她必须在第一支舞开始前，决定自己要抓住哪一种资源。

**节点目标：** 在舞会正厅获得一个立足点：注意力、掩护、情报或人情。

**设计重点：**
- 第三关开始体现前一关入场方式带来的位置差异。
- 所有分支都发生在“舞池边缘 / 正厅外围 / 侧厅入口”范围内，不能跳到完全无关场景。
- 分支要有明确指向：王子注意、身份掩护、秘密情报、社交人情。
- 每个分支都有不超过 1 分钟的演出文本。
- 字池比第二关略复杂，但结果词仍要自然、好懂、有拼字啊哈感。

**核心数值：**
- 体面：决定她是否能被贵族当作同场人物。
- 怀疑：决定她是否会被追问身份。
- 灵感：决定她是否能把偶然变成机会。
- 人情：记录她在舞会里欠下或获得的社交债。

## 第二关到第三关承接演出

根据第二关结果，进入本关前播放一段短演出，并给第三关一个初始倾向。若第二关合成 2 张结果卡，优先播放组合结果中更强势的路线文本，或混合一句。

### 承接 A：名单

**演出文本：**

门卫在她名字旁画下一道短横。女孩走进正厅时，背后还传来纸页翻动的声音。

旁白：她被允许进入。可被允许，不等于被欢迎。名单给她一条路，也把她留在了记录里。

**第三关倾向：**
- 初始怀疑 -1。
- 选择“密信”时更容易被管家阵营注意。

### 承接 B：后门

**演出文本：**

她从侧廊进入，先看见的不是舞池，而是墙边堆放的空酒杯、湿披肩和失物篮。

旁白：从后门进来的人，看到的总是舞会的背面。背面不体面，但藏着很多正面看不见的东西。

**第三关倾向：**
- 初始体面 -1，灵感 +1。
- 选择“密信”或“礼物”时额外获得一条侧厅线索。

### 承接 C：侍女

**演出文本：**

她被安排站在银灯照不到的边缘。有人把空杯递给她，有人把披肩搭到她手臂上，却没人认真看她的脸。

旁白：被忽视并不总是坏事。今晚，忽视可以是一件披在身上的斗篷。

**第三关倾向：**
- 初始怀疑 -1，体面 -1。
- 选择“密信”时效果增强；选择“王子”时怀疑额外 +1。

### 承接 D：舞伴

**演出文本：**

她被带到舞池边，乐声正好亮起来。有人以为她迟到，有人以为她羞怯，还有人已经开始猜她是谁。

旁白：一只伸来的手能带她越过门槛，也能把她推到所有目光中央。

**第三关倾向：**
- 初始体面 +1。
- 选择“王子”或“假面”时效果增强。

## 一、分支剧情

### 分支 A：王子注意

**触发结果卡：** 王子

**结果状态：** 王子注意到女孩。她获得舞会中最高价值的目光，但这份目光会让更多人开始观察她。

**明确指向：** 高机会 / 高关注 / 王子路线

**后续影响：**
- 开启“第一支舞”或“王子询问”节点。
- 若前置带有“水晶鞋”或第二关“舞伴”，本分支体面额外 +1。
- 后续贵族小姐阵营更容易产生敌意。

**数值变化：**
- 体面 +2
- 怀疑 +2
- 灵感 +1

**分支演出文本：**

银灯忽然暗了一瞬，又重新亮起。女孩抬头时，看见舞池另一端的年轻人停下脚步。

他身旁的人还在说话，可他的目光越过人群，落在她身上。

“那位小姐是谁？”他问。

没人立刻回答。这个空白像一支尚未开始的舞曲，安静地把她推到众人视线里。

### 分支 B：假面掩护

**触发结果卡：** 假面

**结果状态：** 女孩获得一层身份掩护。她不需要立刻解释自己是谁，但也很难被真正记住。

**明确指向：** 低怀疑 / 隐藏身份 / 延后暴露

**后续影响：**
- 开启“假面舞步”路线。
- 后续身份盘问延后。
- 王子注意不会立刻强触发，但可通过后续舞步重新获得。

**数值变化：**
- 怀疑 -2
- 灵感 +1
- 体面 +1

**分支演出文本：**

一个贵族小姐笑着经过，随手把面具放在点心桌旁。女孩拿起它时，镜面般的银灯在假面上闪了一下。

她戴上假面。

刚才还盯着她的人，忽然不确定自己看见了什么。她仍在这里，却像被舞会的光藏进了另一层光里。

旁白：看不清，有时比看见更安全。

### 分支 C：密信筹码

**触发结果卡：** 密信

**结果状态：** 女孩得到一封不该出现在舞池边缘的密信。她暂时没有更体面的身份，却握住了能改变谈话方向的筹码。

**明确指向：** 情报 / 谈判 / 侧厅路线

**后续影响：**
- 开启“贵族秘密”路线。
- 后续可与管家、侍女、失物主人或某位贵族交易信息。
- 若第二关走“侍女”或“后门”，本分支灵感额外 +1。

**数值变化：**
- 灵感 +2
- 人情 +1
- 怀疑 +1

**分支演出文本：**

失物篮旁，一封密封信从披肩下面滑了出来。蜡印没有碎，却已经被雨气泡软。

女孩没有拆开它，只看见信封背面写着一个姓氏。

那姓氏刚刚在舞池另一端被人恭敬地念过。

旁白：有些门用请柬打开。有些门，用秘密。

### 分支 D：礼物周旋

**触发结果卡：** 礼物

**结果状态：** 女孩拿到一件可以送出、归还或交换的小礼物。它不一定贵重，但足够制造一次社交接触。

**明确指向：** 人情 / 社交交换 / 中风险

**后续影响：**
- 开启“送还礼物”或“借礼攀谈”节点。
- 后续可选择结交贵族小姐、讨好管家或换取舞池位置。
- 若第一关或第二关怀疑过高，送礼可能被解读为讨好。

**数值变化：**
- 人情 +2
- 体面 +1
- 怀疑 +1

**分支演出文本：**

献礼托盘边缘压着一个没有署名的小盒。盒子不沉，却包着细致的金线。

女孩把它拿在手里，忽然明白：她可以把它送出去，也可以把它还回去。

这两种动作看起来都很体面。

旁白：贫穷的人很少拥有礼物。可今晚，礼物先拥有了她一次机会。

### 1. 王子

**合成配方：**
- 王（来源：王冠纹章）
- 子（来源：金边扇子）

**数值效果：**
- 体面 +2
- 怀疑 +2
- 灵感 +1

**啊哈点：** “王冠纹章”给出“王”，“金边扇子”给出“子”，两个舞会边缘物件拼出最关键的人物“王子”。

### 2. 假面

**卡面图像：** 假花胸针的花影落在点心面包旁，像一张能遮住表情的小面具。

- 假（来源：假花胸针）
- 面（来源：点心面包）

**数值效果：**
- 怀疑 -2
- 灵感 +1
- 体面 +1

**啊哈点：** “假花”的“假”和“面包”的“面”拼成“假面”，从装饰和点心里炼出身份掩护。

### 3. 密信

**卡面图像：** 一封蜡印未破的信从失物篮中露出，信封边缘沾着雨水。

**数值效果：**
- 人情 +2
- 体面 +1
- 怀疑 +1

**啊哈点：** “献礼托盘”给出“礼”，“失物篮”给出“物”，拼成“礼物”，把失物转化成社交资源。

## 三、双结果组合分支

玩家最多可合成 2 张结果卡。若合成 2 张，优先触发组合分支。

### 王子 + 假面：被注意的陌生人

**结果状态：** 王子注意到她，但看不清她是谁。她获得高价值关注，同时暂时保住身份空白。

**追加影响：**
- 开启“神秘舞伴”标签。
- 怀疑本次不再额外增加。
- 后续王子会主动寻找她。

### 王子 + 礼物：借礼靠近

**结果状态：** 女孩借礼物制造与王子或王子身边人的接触。机会更明确，但行为也更容易被旁人解读。

**追加影响：**
- 体面额外 +1。
- 贵族小姐阵营敌意 +1。
- 后续可进入“献礼问话”节点。

### 王子 + 密信：危险筹码

**结果状态：** 她既获得王子注意，又握着一封可能牵涉贵族秘密的信。舞会的浪漫路线立刻带上政治风险。

**追加影响：**
- 开启“王室秘密”标签。
- 灵感额外 +1。
- 后续管家阵营提前介入。

### 假面 + 密信：无名情报人

**结果状态：** 她隐藏身份，同时拿到秘密。她暂时不属于任何圈层，却能把消息送往任何圈层。

**追加影响：**
- 怀疑额外 -1。
- 后续可选择把密信交给管家、贵族小姐或王子。
- 体面不上升。

### 假面 + 礼物：匿名善意

**结果状态：** 她以匿名身份送还或转交礼物。没人知道她是谁，但有人开始欠她一个人情。

**追加影响：**
- 人情额外 +1。
- 怀疑额外 -1。
- 后续可触发“谁帮了我？”事件。

### 密信 + 礼物：交换筹码

**结果状态：** 她同时拥有秘密和礼物，可以选择用礼物包装秘密，或用秘密换取礼物主人的支持。

**追加影响：**
- 开启“交易”标签。
- 人情额外 +1。
- 后续更容易进入贵族支线，但怀疑检查更频繁。

## 六、资源分配感

本关的资源分配重点是：玩家要决定自己在舞会里的第一种资本是什么。

- **王子**：注意力资本。收益最高，风险也最高。
- **假面**：掩护资本。降低怀疑，延后身份问题。
- **密信**：情报资本。不一定体面，但能改变谈判方向。
- **礼物**：人情资本。制造一次社交接触或交换机会。

二卡组合会让资本类型更明确：
- `王子 + 假面`：神秘感
- `王子 + 礼物`：主动接近
- `假面 + 密信`：暗线情报
- `密信 + 礼物`：交易筹码

## 七、剩余字处理示例

基础组句词：**我 / 你 / 在 / 了**

### 示例 1：合成“王子”

**消耗字：** 王 / 子

**剩余字：** 冠 / 纹 / 章 / 金 / 边 / 扇 / 假 / 花 / 胸 / 针 / 点 / 心 / 面 / 包 / 密 / 封 / 信 / 献 / 礼 / 托 / 盘 / 失 / 物 / 篮

**可尝试句：**
- 我在冠纹章金边扇了。
- 假花胸针点心面包在密封信献礼托盘失物篮。

**预期反馈：**
- 只抓王子会留下大量场景杂物，负担较高。
- 教学意义：最高价值目标不是免费获得的，剩余词会把舞会压力压回来。

### 示例 2：合成“假面 + 密信”

**消耗字：** 假 / 面 / 密 / 信

**剩余字：** 王 / 冠 / 纹 / 章 / 金 / 边 / 扇 / 子 / 花 / 胸 / 针 / 点 / 心 / 包 / 封 / 献 / 礼 / 托 / 盘 / 失 / 物 / 篮

**可尝试句：**
- 王冠纹章金边扇子在我了。
- 花胸针点心包封献礼托盘失物篮在你。

**预期反馈：**
- 合理分中低，怀疑负担较低。
- 适合隐藏身份并进入情报路线。

### 示例 3：合成“密信 + 礼物”

**消耗字：** 密 / 信 / 礼 / 物

**剩余字：** 王 / 冠 / 纹 / 章 / 金 / 边 / 扇 / 子 / 假 / 花 / 胸 / 针 / 点 / 心 / 面 / 包 / 封 / 献 / 托 / 盘 / 失 / 篮

**可尝试句：**
- 我在王冠纹章金边扇子。
- 假花胸针点心面包封献托盘失篮了。

**预期反馈：**
- 抽象分高，负担中。
- 适合走交易路线，但“王子”和“假面”的关键字都被剩下，意味着主动放弃高光和掩护。

- [x] 每个分支有不超过 1 分钟的演出文本。
- [x] 所有分支剧情都符合“银灯舞池 / 正厅边缘”场景。
- [x] 结果卡像自然词或自然短语：王子、假面、密信、礼物。
