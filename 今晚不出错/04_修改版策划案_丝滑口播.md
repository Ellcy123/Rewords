# 《今晚不出错》修改版策划案 —— "丝滑口播"系统

> 版本：v2.0（覆盖并扩展 01/02 中"口播插入"相关章节，其余系统不变）
> 日期：2026-07-03
> 状态：待开发。本案包含完整机制规格、数据结构变更、第一关全部内容。

---

## 一、修改动机（为什么改）

当前口播玩法只有一个维度：**赶在倒计时结束前，把卡拖进任意一个段落间隙**。所有间隙等价、插哪都一样，玩家玩两轮就发现"这只是个限时快递任务"。

修改目标，来自三条设计判断：

1. **纠错是节拍器，口播是升压器。** 纠错从头到尾保持"点词→三选一"的固定动作，提供稳定底层节奏；难度曲线和压力叠加全部由口播系统承担——卡越发越多、越叠越密。
2. **口播考的是"节奏 + 语义"，不是手速。** 参考主播圈的"丝滑口播"梗（"我想出门转转——诶，说到转转，旧手机回收就上转转"）：广告从正文的某个词无缝拐出去，拐得越自然越是本事。玩家要在流动的提词器文本里**同时扫描两种目标**：错误词（点击纠错）和钩子词（拍卡接广告）。
3. **时机窗口带来节奏天国式手感。** 钩子词会被光标碾过去，早了晚了都不行。盯着光标流速、在窗口内出手，这是文字版的打拍子。

## 二、核心机制：钩子词口播（Hook-Word Segue)

### 2.1 三种插入方式、四档判定

每张口播卡带 1~2 个**钩子词**（印在卡面上）。玩家可以：

| 档位 | 操作 | 判定条件 | 收视率 | 演出反馈 |
|---|---|---|---|---|
| **丝滑** | 选中卡后，光标进入钩子窗口时按 `V`（或点击卡面） | 窗口命中 + 段落语境安全 | **+8** | 主播顺着正文念出接梗词，金色闪光，弹幕刷"老无缝了""这广告接得比新闻还顺" |
| **稳妥** | 拖卡进段落间隙（现有玩法，完整保留） | 落在任意间隙 | **+3** | 正常口播段，弹幕"广告插得很端庄" |
| **尬插** | 选中卡后在非钩子词上按 `V` / 把卡拖到句子中间 | 窗口外强插 | **−5** | 主播念到一半突然拐进广告再硬拐回来，红色闪光，弹幕爆炸 |
| **血压** | 钩子命中但段落语境为"沉重" | 窗口命中 + 语境冲突 | **−8** | 灾害新闻后接矿泉水广告，全场弹幕"甲方看了都沉默"，导播咆哮 |
| （超时） | 卡片倒计时归零 | — | −6 | 现有逻辑不变 |

**设计要点：**
- 稳妥档永远存在，保证低水平玩家有安全出口；丝滑档奖励是它的 2.6 倍，拉开追求空间。
- 血压档是本系统的隐藏牙齿：钩子词匹配 ≠ 可以插。暴雨段落里出现"水"字，正是给蓝天矿泉水卡挖的坑。它教玩家"读语境，不是找关键字"，也是最容易出传播截图的瞬间。

### 2.2 时机窗口（节奏判定）

```
句子： 网友 调侃 ： 想 偶遇 不用 蹲 直播 ， 出门 [转转] 就行 。
                                        ↑窗口开   ↑窗口关
```

- **窗口开启**：光标进入钩子词的**前一个实词** token 时；
- **窗口关闭**：钩子词 token 念完时；
- 按现有 token 时长公式（`max(0.34, 字数/光标速度)` ），窗口约 **1.2~2.2 秒**，与纠错弹窗 3.5 秒错开量级，形成"短窗口拍卡、长窗口选词"两种节奏。
- 窗口期内播放轻微的"预备"音效（复用 softTick，音高提升），命中时播 ding + 金色 token 闪光——听觉上就是节奏游戏的 cue → hit。

### 2.3 丝滑插入的演出流程

1. 光标念到钩子词，玩家按 `V`；
2. 钩子词 token 变金色，其后**动态拼入接梗句 token**（复用现有 `insertRuntimeSponsorParagraph` 的 token 拼接思路，但拼在句内而非段后）；
3. 主播读完接梗句（约 3~5 秒，期间不发新错误，给玩家喘息），光标**跳回原句断点**继续；
4. 结算 +8，卡片消失。

接梗句模板存在卡数据里：`"——诶，说到{hook}，{adText}——好，回到新闻。"`

### 2.4 钩子词与错误词的复合冲突（第三幕高潮设计）

同一个词可以**既是错误又是钩子**。例：稿子写"市民纷纷出门**赚赚**"（错别字，正确为"转转"），而你手里压着转转卡：

- 直接拍卡 → 主播念"说到赚赚……"，判**尬插**（接了个错词）；
- 先点击纠错、三选一改对 → 改对的瞬间该 token 重读一次，**重读即窗口**，此时拍卡 → 丝滑 + 纠错双结算（+3 +8），全场最高光操作；
- 犹豫 → 窗口过去，钩子作废，只剩纠错分。

这个复合节点每幕最多埋 1 个，是给高手准备的"满分谱面"。

## 三、叠加曲线（口播压力如何逐幕升级）

| 幕 | 手牌上限 | 发卡节奏 | 钩子密度 | 新压力源 |
|---|---|---|---|---|
| 第一幕 | 1 张 | 开播 20s 发第 1 张，无死线 | 每段 1 个钩子，卡面+提词器双端淡金下划线提示 | 教学：间隙插入 与 丝滑插入 各保底体验一次 |
| 第二幕 | 2 张 | 每 45s 发 1 张，死线 40s | 钩子不再高亮，仅卡面印钩子词 | 台规开始占用注意力；出现第一个"语境陷阱"段 |
| 第三幕 | 3 张 | 每 30s 发 1 张，死线 30s；含 1 张突发插报 | 钩子稀疏（隔段才有），部分钩子在死线之后 | 复合冲突词 ×1；玩家必须取舍：超时认罚 / 硬插间隙 / 喝水拖时间等钩子 |

**喝水的战略升级**：暂停 3 秒 = 把即将错过的钩子窗口"冻"住，或者把死线前的最后一个间隙拖到手。自救道具从"救命"变成"控节奏"，不需要改任何代码，是机制的自然涌现。

## 四、数据结构变更

### 4.1 cards.json（每张卡新增 3 个字段）

```json
{
  "id": "card_zhuanzhuan",
  "type": "sponsor",
  "name": "转转严选·二手回收",
  "hooks": ["转转", "转一转"],
  "segueTemplate": "——诶，说到{hook}，旧手机回收就上转转严选，多卖三百块——好，回到新闻。",
  "gapReadText": "本节目由转转严选特约播出，旧手机回收就上转转严选。",
  "deadline": 40,
  "rewards": { "smooth": 8, "onTime": 3, "awkward": -5, "clash": -8, "timeout": -6 }
}
```

### 4.2 script.json（段落新增 1 个字段，句子新增 1 个可选字段）

```json
{
  "id": "a1p3",
  "title": "汛期天气",
  "mood": "heavy",          // "light" | "neutral" | "heavy"；heavy 段落内钩子命中判"血压"
  "sentences": [
    {
      "tokens": ["水利部", "预计", "今天", "起", "全面", "进入", "主汛期", "，", "..."],
      "hookSeeds": [{ "tokenIndex": 0, "note": "『水』字陷阱，供关卡检查用" }]
    }
  ]
}
```

> `hookSeeds` 仅作关卡设计自查标注，运行时匹配以 **token 文本 ∈ 当前手牌 hooks** 为准，无需预绑定——这样同一张卡在任何含钩子词的稿件里都能用，内容扩展零成本。

### 4.3 app.js 改动清单（预估工作量：半天）

| 模块 | 改动 |
|---|---|
| 卡片选中态 | 已有 `selected`；新增：选中含 hooks 的卡时，进入"待拍"模式 |
| 窗口判定 | 光标推进函数里，若 `当前token或下一token文本 ∈ 选中卡.hooks` → 开窗标记；`V` 键 / 点击卡面时查窗口标记 → 四档分流 |
| 丝滑演出 | 新函数 `spliceSegueTokens(card, tokenIndex)`：金色化钩子 token，句内拼入 segue tokens，读完跳回断点（复用 midSentence 的拼接管线，改结果分档） |
| 语境判定 | `paragraph.mood === "heavy"` 且窗口命中 → clash 分支 |
| 复合冲突 | 纠错改对后，该 token 触发 0.8s 重读窗口（新增一个 `rereadUntil` 时间戳即可） |
| UI | 卡面渲染 hooks 徽章；第一幕钩子词 token 加 `hook-hint` 下划线类；V 键提示 |
| 反馈 | 四档各接一条 `showFeedback` + 专属弹幕池（danmaku.json 加 `smooth` / `clash` 两个池） |

间隙插入、超时、纠错、台规、收视率系统**全部不动**。

### 5.1 第一幕《民生暖场》（mood: light，教学幕）

**段落 a1p1「企业家过早」**（改编自：雷军武汉大成路吃早餐上热搜）

> 「最近，某手机公司创始人现身武汉过早一条街，排队吃了一碗〖螺蛳粉〗，引来大批市民围观拍照。」
> - 错误：〖螺蛳粉〗→ 正确「热干面」；干扰项：豆皮 / 螺蛳粉（保留）／热干面
>   - 知识点：武汉过早 = 热干面；螺蛳粉是柳州的
> 「本人回应说，吃个早饭上了好几天热搜，自己也匪夷所思，流量时代大家笑一笑就好。」（无错误，纯念稿，练光标跟读）
> 「网友调侃：想偶遇不用蹲直播间，出门〔转转〕就行，说不定就碰上了。」
> - **钩子：〔转转〕**——C1 教学窗口，此段前系统弹导播提示："手里那张卡的钩子词快到了，光标碾到它之前按 V"

**段落 a1p2「暑期福利」**（改编自：多景区对中高考生免票）

> 「四川多个景区宣布，暑假期间对今年参加中高考的学生实行免票或半价，凭〖录取通知书〗即可入园。」
> - 错误：〖录取通知书〗→ 正确「准考证」；干扰项：学生证 / 准考证 / 录取通知书
>   - 知识点：7 月初大部分录取通知书还没发，景区核验的是准考证（常识+时效推理，比纯知识题高半档）
> 「不少家长表示，孩子考完正好出去走走，放松一下。」（无错误）
>
> 【段落间隙】←教学：若玩家没接住 C1 的钩子，导播提示"拖到这个空档里也行，稳妥拿分"

### 5.2 第二幕《科技快讯》（mood: neutral → 首个陷阱段 heavy）

**段落 a2p1「一箭十八星」**（改编自：千帆星座一箭 18 星）

> 「昨天凌晨，我国成功实施一箭〖八十一〗星发射任务，千帆星座在轨卫星数量增至两百颗。」
> - 错误：〖八十一〗→ 正确「十八」；干扰项：十八 / 一百零八 / 八十一
>   - 知识点：数字对调型错误，本关最典型的"看着顺嘴念过去就完了"
> 「按照规划，该星座未来将为全球用户提供低轨宽带服务，手机直连卫星不再是科幻。」（无错误）
> - **钩子：无**（本段考纯纠错，让玩家体验"手里压着卡但没地方丝滑"的痒感）

**段落 a2p2「充电新政」**（改编自：十部门高速服务区充电设施新政）

> 「十部门要求，到 2028 年底前，高速公路服务区新建改建六十千瓦以上功率的电动汽车〔充电〕设施三万个。」
> - **钩子：〔充电〕**——C3 的丝滑窗口
> 「今后跑长途，服务区喝杯茶的工夫就能〔充电〕到八成，里程焦虑会越来越少。」
> - **钩子：〔充电〕二次机会**——第一个窗口错过还有备份，教学"钩子可能不止一个"
> - 台规发放：〖禁字令：不许说"四"〗（下一段有"四"字埋伏）

**段落 a2p3「主汛期预警」**（mood: **heavy**，语境陷阱段）

> 「水利部预报，七月一日起全国全面进入主汛期，北方洪涝偏重，局地有极端暴雨洪〖旱〗。」
> - 错误：〖旱〗→ 正确「水」；干扰项：水 / 灾 / 旱
>   - 双重陷阱：正确答案"水"恰好是 C2 蓝天矿泉水的钩子词！纠错改对后 token 重读，**此时拍卡 = 血压判定 −8**（灾害段落）。改对但忍住不拍，才是正解。这是全关最重要的一课：**钩子匹配 ≠ 可以插**。
> 「黔渝湘鄂等地部分地区有大暴雨，请当地观众留意预警信息，减少外出。」（无错误）
> - 若 C2 还在手上：本段之后的间隙同样判血压吗？——**不判**。间隙插入自带"我们先休息一下"的缓冲话术，永远安全，只拿基础分。这保证陷阱惩罚的是"贪丝滑"，不是"拿着卡倒霉"。

**段落 a2p4「高温提示」**（mood: light，C2 的正确归宿）

> 「南方多地今天最高气温超过三十七度，户外工作者请注意防暑，及时补〔水〕。」
> - **钩子：〔水〕**——同一个字，换个段落就是满分位。前后两段构成本关的核心教学对照。

### 5.3 第三幕《晚间冲刺》（发卡加速，复合冲突登场）

**段落 a3p1「CBA 夺冠」**（改编自：上海男篮时隔 24 年夺冠）

> 「时隔〖四十二〗年，上海男篮再次捧起 CBA 总冠军奖杯，全城球迷彻夜庆祝。」
> - 错误：〖四十二〗→ 正确「二十四」；干扰项：二十四 / 三十四 / 四十二
>   - 复合压力：此时〖禁字令：不许说"四"〗生效中——但"二十四"含"四"！本设计为**台规与纠错的对撞**：正确操作是选"二十四"（纠错优先级高于台规，台规对纠错结果免罚，规则页注明），考玩家敢不敢按下去。
> 「有球迷说，等这一天等得头发都白了。」（无错误）

**段落 a3p2「医保新政 + 复合冲突」**（改编自：医保个账跨省共济）

> 「职工医保个人账户实现跨省共济，配偶、父母、子女都能用，异地帮爸妈买药更方便了。」（无错误，但语速加快 10%）
> 「专家提醒，绑定操作在手机上就能完成，不会弄的可以让〖孩纸〗帮忙。」
> - **复合冲突词：〖孩纸〗**→ 正确「孩子」，而「孩子」是 C4 金榜学习机的钩子词！
>   - 满分操作：点击纠错 → 选"孩子" → 重读窗口 0.8s 内按 V → +3 +8，弹幕刷屏"这波是主播圈的博尔特"
> 
> 【间隙】← C5 突发插报（台风路径更新）在此前 25s 发放，只能走间隙，挤占玩家规划

**段落 a3p3「收尾」**（mood: light）

> 「今晚的节目就到这里。天气多变，出门记得带伞；手机换代别忘了旧机回收；祝各位考生金榜题名。」
> - 彩蛋：这句话里埋着本关所有卡的钩子词残影（转转→回收、学习机→金榜），若还有卡没打出去，这是最后的丝滑窗口——念完即结算，超时卡全部按超时扣分。

## 六、开发顺序建议（在现有代码上，总计约 1.5 天）

1. **数据先行**（2h）：cards.json 加 hooks/segueTemplate/rewards 四档；script.json 换第一关内容 + mood 字段；danmaku.json 加三个池。
2. **窗口判定 + V 键**（3h）：光标推进时对选中卡做 hooks 文本匹配，开关窗口标记；`V` 分流四档。
3. **丝滑演出**（3h）：句内拼接 segue tokens、金色闪光、跳回断点；复用 midSentence 管线。
4. **陷阱与复合**（2h）：mood=heavy 的 clash 分支；纠错后 0.8s 重读窗口。
5. **教学提示 + 卡面钩子徽章 + 反馈接线**（2h）。
6. **调参试玩**（剩余时间）：重点盯窗口时长（1.2s 是否太苛刻）和第三幕手牌压力（3 张是否过载）。

## 七、风险与对策

| 风险 | 对策 |
|---|---|
| 玩家没看卡面，不知道钩子词是什么 | 第一幕双端高亮 + 导播口头教学；卡面钩子词用大号金字 |
| 窗口太短，感觉像 QTE 而不是节奏 | 窗口覆盖"前一实词+钩子词"两个 token；预备音效提前 cue；不行就再纳入前两个词 |
| 血压陷阱被骂"阴人" | 陷阱段落有明显视觉信号（提词器边框转暗红 + 段落标题带⚠），惩罚的是没读语境，不是没读攻略 |
| 丝滑收益太高，间隙沦为摆设 | 死线和钩子错位设计保证约 1/3 的卡"等不到钩子"，间隙是它们的唯一出路 |
