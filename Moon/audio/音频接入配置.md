# Moon 音频接入配置

> 给程序接入用。音效位于 `Moon/audio/sfx/`，关卡音乐位于 `Moon/audio/music/`。大部分音效为 MP3，敲门音效 `door_knock` 为本地合成 WAV，用来保证只有一下且音量足够。

## 接入原则

- `id` 是游戏内事件名，建议直接作为音频 key。
- SFX 的 `file` 是相对 `Moon/audio/sfx/` 的文件名。
- BGM 的 `file` 是相对 `Moon/audio/music/` 的文件名。
- SFX 除 `clock_tick_loop` 外，全部是一次性播放。
- BGM 默认循环播放，进入关卡时淡入，离开关卡时淡出。
- 错误反馈音效不需要打断当前动画，跟随机关动画开始播放即可。
- 正确反馈音效建议在机关动作开始时播放，成功 UI 提示可以晚 0.1~0.2 秒。

## 关卡音乐

| id | file | loop | 建议音量 | 用途 |
|---|---|---:|---:|---|
| `level01_bgm` | `level01__stage_door_taps.mp3` | 是 | `0.55` | 第 1 关：开演前的门与梯子。木门、吊梯、教学关，轻快但不抢戏。 |
| `level02_bgm` | `level02__backstage_moon_puzzle.mp3` | 是 | `0.55` | 第 2 关：月亮、幕布与挂钟。后台、挂钟、幕布机关感。 |
| `level03_bgm` | `level03__trapdoor_tinker.mp3` | 是 | `0.55` | 第 3 关：搭建卷尺发射台。最终机关区，卷尺、跷跷板、沙袋，节奏更有推动感。 |

## 关卡 01：开演前的门与梯子

| id | file | loop | 触发时机 |
|---|---|---:|---|
| `door_knock` | `level01__door_knock.wav` | 否 | 关卡开场，小演员在门外慢慢敲门，只敲一下。 |
| `door_wrong_hit` | `level01__door_wrong_hit.mp3` | 否 | 门锚点错误，玩家点击门，木门磕碰/拍到小演员。 |
| `door_open` | `level01__door_open.mp3` | 否 | 门锚点在左侧，玩家点击门，门正常打开。 |
| `ladder_wobble_fail` | `level01__ladder_wobble_fail.mp3` | 否 | 梯子锚点在中间，玩家点击梯子，梯子下压回弹。 |
| `ladder_wrong_hit` | `level01__ladder_wrong_hit.mp3` | 否 | 梯子锚点在左端，玩家点击梯子，梯子扫到小演员。 |
| `ladder_land` | `level01__ladder_land.mp3` | 否 | 梯子锚点在右端，玩家点击梯子，梯子落成可爬斜梯。 |

## 关卡 02：月亮、幕布与挂钟

| id | file | loop | 触发时机 |
|---|---|---:|---|
| `moon_wobble` | `level02__moon_wobble.mp3` | 否 | 月亮布景被点击后晃动。默认、错误、正确偏倒都可先共用。 |
| `clock_tick_loop` | `level02__clock_tick_loop.mp3` | 是 | 进入第 2 关后循环播放；过关或离开关卡时停止。 |
| `clock_pivot_turn` | `level02__clock_pivot_turn.mp3` | 否 | 挂钟锚点改到右侧后，玩家点击挂钟，钟面转向演员。 |
| `minute_hand_jam` | `level02__minute_hand_jam.mp3` | 否 | 分针锚点错误，点击分针时卡住/轻抖。 |
| `wake_slap_switch` | `level02__wake_slap_switch.mp3` | 否 | 分针锚点在下端且角度正确，点击分针，拍醒演员并触发开关。 |
| `curtain_bunch_fail` | `level02__curtain_bunch_fail.mp3` | 否 | 幕布锚点在中间，拉幕机关启动后幕布被拉成一团。 |
| `curtain_slide_open` | `level02__curtain_slide_open.mp3` | 否 | 幕布锚点在左侧，玩家点击幕布/开关，幕布正确拉开。 |

## 关卡 03：搭建卷尺发射台

| id | file | loop | 触发时机 |
|---|---|---:|---|
| `triangle_lift_fail` | `level03__triangle_lift_fail.mp3` | 否 | 三角形左侧默认锚点，轻微抬起又落回斜面。 |
| `triangle_roll_fulcrum` | `level03__triangle_roll_fulcrum.mp3` | 否 | 三角形右侧正确锚点，沿斜面翻滚并卡到木板支点位。 |
| `tape_retract_jump_fail` | `level03__tape_retract_jump_fail.mp3` | 否 | 卷尺尺盒锚点错误，尺带回收后尺盒原地跳/转。 |
| `tape_slide_onto_plank` | `level03__tape_slide_onto_plank.mp3` | 否 | 卷尺尺头锚点正确，尺盒被拉出并滑上木板发射位。 |
| `sandbag_weak_creak` | `level03__sandbag_weak_creak.mp3` | 否 | 沙袋顶部/中间锚点错误，只轻晃、嘎吱、掉一点灰。 |
| `sandbag_swing_dirt_fall` | `level03__sandbag_swing_dirt_fall.mp3` | 否 | 沙袋底部锚点正确，大幅摆动并震落泥土到木板右端。 |
| `seesaw_launch_switch` | `level03__seesaw_launch_switch.mp3` | 否 | 泥土压板后跷跷板弹起，卷尺飞向高处开关并撞开。 |
| `tape_rebound_bonk_finale` | `level03__tape_rebound_bonk_finale.mp3` | 否 | 卷尺回弹砸到小演员，同时灯亮/幕布开，最终喜剧收尾。 |

## 事件绑定建议

```js
// 伪代码
playMusic("level01_bgm");
playMusic("level02_bgm");
playMusic("level03_bgm");
stopMusic();

playSfx("door_knock");
playSfx("door_wrong_hit");
playSfx("door_open");
playSfx("ladder_wobble_fail");
playSfx("ladder_wrong_hit");
playSfx("ladder_land");

playLoop("clock_tick_loop");
stopLoop("clock_tick_loop");
playSfx("moon_wobble");
playSfx("clock_pivot_turn");
playSfx("minute_hand_jam");
playSfx("wake_slap_switch");
playSfx("curtain_bunch_fail");
playSfx("curtain_slide_open");

playSfx("triangle_lift_fail");
playSfx("triangle_roll_fulcrum");
playSfx("tape_retract_jump_fail");
playSfx("tape_slide_onto_plank");
playSfx("sandbag_weak_creak");
playSfx("sandbag_swing_dirt_fall");
playSfx("seesaw_launch_switch");
playSfx("tape_rebound_bonk_finale");
```

## 源文件

- 生成 manifest：`Moon/audio/sfx/sfx_manifest.json`
- 程序配置 JSON：`Moon/audio/audio_config.json`
- 音乐目录：`Moon/audio/music/`
- 第 1 关需求：`Moon/关卡01_精简音效需求.elevenlabs.json`
- 第 2 关需求：`Moon/关卡02_精简音效需求.elevenlabs.json`
- 第 3 关需求：`Moon/关卡03_精简音效需求.elevenlabs.json`
- 合集需求：`Moon/关卡01-02_精简音效需求合集.elevenlabs.json`
