# W101 二次元反差色风格测试设计

> 日期：2026-07-18
>
> 状态：方向已确认，待样图实施

## 1. 目标与范围

为 `W101《有梯子，但不会修》` 制作一张 9:16 二次元风格样图，用来判断新的项目美术方向是否比现有拟真人试片更合适。

本轮只生成单张静态样图，不调用 RunningHub、不生成视频、不替换现有 W001/W101 资产，也不修改剧情或字幕配置。

## 2. 视觉方向

采用高饱和霓虹互补色与清晰赛璐璐上色：

- 主阴影为深青蓝和蓝紫；
- 灯架失稳处使用橙红危险光；
- 洋红只用于局部边缘光和婚礼装饰；
- 新娘婚纱保留象牙白，避免整体变成霓虹夜店；
- 角色使用干净轮廓、两至三阶明暗和略夸张表情；
- 场景保留日间婚礼的可读性，同时通过冷暖冲突制造事故感。

画面应像高完成度原创动画关键帧，不模仿特定在世艺术家的个人画风，也不直接复刻某一现成动画角色。

## 3. 构图与叙事

沿用 W101 已验证的叙事构图：

- 新娘位于画面中下部，是情绪中心；
- 完整银色梯子从地面延伸至左上区域；
- 伴郎站在梯子上，伸手误动灯架零件；
- 黑色灯架斜压画面上方，橙红警示光集中在松脱卡扣附近；
- 宾客后退或抬手惊呼，但不喧宾夺主；
- 通过速度线、碎屑和高对比边缘光表达即将坠落，不表现血腥或直接撞击。

首屏必须同时读出“婚礼”“梯子上的外行”“灯架危险”三个信息。

## 4. 输出

样图保存为：

`刷到你了/assets/style-tests/W101/W101_anime_contrast_v1.png`

要求：

- 9:16 竖屏 PNG；
- 无字幕、平台 UI、Logo 或水印；
- 不覆盖现有拟真人关键帧；
- 人物、梯子和灯架结构无影响理解的严重畸形。

## 5. 验收标准

- 冷青蓝与暖橙红形成明确反差，洋红只做点缀；
- 婚纱仍为视觉焦点，人物肤色没有被霓虹完全染色；
- 二次元赛璐璐质感明确，不像加滤镜的真人照片；
- 梯子完整、伴郎动作明确、灯架危险关系一眼可读；
- 具备短视频首帧冲击力，并适合后续测试图生视频的一致性；
- 无文字、血腥、直接撞击或品牌元素。

## 6. 后续决策

本轮只判断静态风格。样图通过后，再单独决定是否用同一画风重做 W001、W101 视频测试；样图不通过则调整配色或轮廓语言，不消耗 RunningHub 点数。
