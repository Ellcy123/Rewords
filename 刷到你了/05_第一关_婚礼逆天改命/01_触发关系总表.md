# 第一关《婚礼逆天改命》触发关系总表

> 本文是第一关节点触发关系的唯一事实来源。剧情文件只描述视频内容，不另行维护完整配对表。

## 一、初始状态

关卡开始时，推荐流固定存在：

- W001《婚礼灯架事故》；
- C001《王妃翻墙私逃》；
- K001《空调开十六度，电脑会变快吗？》。

玩家初始背包为空，但拥有足够完成首次教学购买的货币。

初始可购买内容：

| 来源节点 | 挂载链接 | 购买结果 |
|---|---|---|
| C001 | 刺客同款多功能梯子 | 获得 1 份“梯子” |
| K001 | 同城空调师傅上门一次 | 获得 1 份“空调师傅” |

商品链接首次解锁后永久进入“已发现商品”。玩家可以重复购买，以便把同一种物品投进不同视频。每次成功投放消耗 1 份对应物品。

## 二、资源树触发

资源树负责取得后半段婚礼所需的录音笔与投影服务。

| 编号 | 触发表达式 | 新链接 | 作用 |
|---|---|---|---|
| R01 | K001 + 梯子 → K101《梯子变成 VPN》 | 录音笔 | 通过“梯子／翻墙”黑话取得取证工具 |
| R02 | C001 + 录音笔 → C101《金銮殿播放录音》 | 投影服务 | 通过古装硬广取得公开证据的服务 |

完整资源路径依次经过 R01、R02；两次投放之间，玩家分别从 K101 购买录音笔，并在 C101 解锁投影服务。

这条路径可以在婚礼主线推进前完成，也可以在玩家遇到 W300、W301 后再回来完成。

## 三、婚礼主线触发

| 编号 | 触发表达式 | 婚礼状态 |
|---|---|---|
| W-R01 | W001 + 梯子 → W101《有梯子，但不会修》 | 新娘仍死亡 |
| W-R02 | W101 + 空调师傅 → W200《新娘活下来了》 | 新娘存活，灯架问题解决 |
| W-R03 | W200 首次完播 → W300《婚礼当天私会维修工？》 | 谣言出现，婚礼取消 |
| W-R04 | W300 + 录音笔 → W301《证据就在笔里，但没人看见》 | 证据存在，现场仍不相信 |
| W-R05 | W301 + 投影服务 → W400《证据上大屏，婚礼完成》 | 婚礼顺利结束 |

婚礼线的推进顺序固定为 W-R01、W-R02、W-R03、W-R04、W-R05，其中 W-R03 是完播事件，其余四项是物品投放。

W200 完播生成 W300，是同一场婚礼中的人物发布后续，不属于系统按问题推送商品，也不消耗物品。

## 四、节点解锁条件

| 节点 | 解锁条件 | 是否可作为正向投放目标 |
|---|---|---|
| W001 | 开局 | 是，接收梯子 |
| C001 | 开局 | 是，接收录音笔 |
| K001 | 开局 | 是，接收梯子 |
| K101 | K001 首次成功接收梯子 | 否，仅提供录音笔链接 |
| C101 | C001 首次成功接收录音笔 | 否，仅提供投影服务链接 |
| W101 | W001 首次成功接收梯子 | 是，接收空调师傅 |
| W200 | W101 首次成功接收空调师傅 | 否，完播生成 W300 |
| W300 | W200 首次完整播放结束 | 是，接收录音笔 |
| W301 | W300 首次成功接收录音笔 | 是，接收投影服务 |
| W400 | W301 首次成功接收投影服务 | 否，关卡完成节点 |

## 五、生成与重复规则

1. 成功投放后，立即播放对应的预制结果视频；
2. 新视频播放结束后加入正常推荐流；
3. 同一个“来源视频 + 物品”正向组合只生成一次；
4. 重复执行已完成的组合不消耗物品，也不生成第二份节点；
5. 来源视频继续留在推荐流，直到阶段收束或玩家主动隐藏；
6. 已解锁链接不会因来源视频被隐藏而消失；
7. W400 播放完毕后结算关卡，不再生成新的必需节点。

## 六、本轮错误配对处理

本轮文档只定义可通关树。没有列入上表的配对，不视为无效设计，而是留给下一轮荒诞终局矩阵。

在荒诞终局矩阵完成前，原型中的未定义配对应显示为不可选择，避免临时的“无效礼物”反馈成为正式规则。
