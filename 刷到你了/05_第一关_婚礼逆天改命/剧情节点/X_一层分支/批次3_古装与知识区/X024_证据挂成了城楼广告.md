# X024《证据挂成了城楼广告》

- 触发来源：C101 + 梯子；
- 结果类型：另一种人生终局；
- 总时长：9 秒；
- Meta 强度：无；
- 生长状态：终局，不再继续生长；
- 新商品链接：无。

### 0～1 秒：先给广告结果

- 城楼上挂着录音笔，牌子写：**“国师亲口认证，翻墙首选。”**

### 1～5 秒：把证据挂得太远

- 王妃爬梯子把录音笔挂到城楼最高处；
- 她喊：“这样全城都能看见！”
- 城下百姓只看见一支黑笔和巨幅广告；
- 国师趁机说：“王妃在带货。”

### 5～9 秒：冤案变广告业

- 商贩抢着买城楼广告位；
- 王妃收下第一笔租金：“证据没人看，广告有人看。”
- 快切：**“大唐城墙广告总监。”**

## 终局状态

王妃放弃庭审，经营城楼广告位。本节点进入命运记录，不再继续生长。
