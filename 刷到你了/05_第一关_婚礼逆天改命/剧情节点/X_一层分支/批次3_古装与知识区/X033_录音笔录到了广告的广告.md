# X033《录音笔录到了广告的广告》

- 触发来源：K101 + 录音笔；
- 结果类型：另一种人生终局；
- 总时长：10 秒；
- Meta 强度：无；
- 生长状态：终局，不再继续生长；
- 新商品链接：无。

### 0～1 秒：先给嵌套结果

- 录音笔播放广告：“本广告由录音笔广告赞助。”
- 博主沉默：“广告开始自我繁殖了。”

### 1～5 秒：录下比内容更多的广告

- 博主拿录音笔录海外网页；
- 页面广告的声音里又插入一条广告；
- 每录一层，下一层都说“跳过前请看完”；
- 跑分数据只闪过零点一秒。

### 5～10 秒：把干扰变成 ASMR

- 博主把十层广告叠成节奏音轨；
- 快切频道：**“广告 ASMR：跳不过就听睡。”**
- 视频结尾自动播出下一条广告，博主笑着关不掉。

## 终局状态

博主放弃知识内容，转行制作广告嵌套 ASMR。本节点进入命运记录，不再继续生长。
