# X027《影壁里多出一个皇帝》

- 触发来源：C101 + 投影服务；
- 结果类型：Meta 终局；
- 总时长：12 秒；
- Meta 强度：中度；
- 生长状态：终局，不再继续生长；
- 新商品链接：无。

### 0～1 秒：先给双皇帝结果

- 影壁上的投影皇帝先开口：**“评论说你断案太慢。”**
- 现实皇帝回头：“谁在朕墙里？”

### 1～5 秒：投影比本人先看见弹幕

- 投影服务把庭审投上影壁；
- 影壁里的皇帝比现实动作快半拍，还能读到飘过的评论；
- 他替王妃宣布：“国师淘汰。”
- 现实皇帝急了：“朕还没说！”

### 5～12 秒：影壁抢走皇位

- 宫女纷纷向影壁皇帝跪拜，因为他会回评论；
- 现实皇帝只好改做影壁运营；
- 王妃站在旁边宣布新栏目：**“影壁皇帝在线断案。”**
- 最后评论：“这位回复快，建议常驻。”

## Meta 规则

只有影壁中的皇帝看得到评论；它是投影异常，不解释来源。

## 终局状态

现实皇帝失去庭审主导权，王妃运营影壁断案节目。本节点进入命运记录，不再继续生长。
