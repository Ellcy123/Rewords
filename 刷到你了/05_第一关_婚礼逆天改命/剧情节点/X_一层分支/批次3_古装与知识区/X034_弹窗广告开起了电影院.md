# X034《弹窗广告开起了电影院》

- 触发来源：K101 + 投影服务；
- 结果类型：Meta 终局；
- 总时长：11 秒；
- Meta 强度：轻度；
- 生长状态：终局，不再继续生长；
- 新商品链接：无。

### 0～1 秒：先给影院结果

- 墙上巨幕播放弹窗广告，座位坐满观众；
- 广告角落显示：**“当前观看人数：1,000,000。”**

### 1～5 秒：弹窗知道自己被看

- 博主把海外网页投到墙上；
- 一个弹窗广告被放大成电影画面；
- 广告里的演员看向镜头：“人这么多，能别点跳过吗？”
- 观众真的把手机放下。

### 5～11 秒：知识区改做广告影院

- 博主试图找跑分数据，所有弹窗排队买放映档期；
- 他拿起售票器：“第一排看会员广告，后排看加载页。”
- 快切影院招牌：**“弹窗国际电影节。”**
- 最后一个广告字幕：“感谢你观看广告的观看人数。”

## Meta 规则

广告角色只在这条投影时间线中意识到自己被观看，不与其他 Meta 节点连通。

## 终局状态

博主把知识频道改为弹窗广告影院。本节点进入命运记录，不再继续生长。
