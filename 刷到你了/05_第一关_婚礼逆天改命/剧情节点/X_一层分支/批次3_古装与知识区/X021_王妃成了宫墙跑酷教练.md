# X021《王妃成了宫墙跑酷教练》

- 触发来源：C001 + 梯子；
- 结果类型：另一种人生终局；
- 总时长：10 秒；
- Meta 强度：无；
- 生长状态：终局，不再继续生长；
- 新商品链接：无。

## 分秒脚本

### 0～1 秒：先给职业结果

- 王妃从宫墙飞身落地，身后跟着一排拿梯子的宫女；
- 大字：**“冷宫王妃，转行宫墙跑酷教练。”**

### 1～5 秒：梯子太多反而像赛道

- 国师刚否认翻墙，王妃又掏出三把梯子；
- 宫女问：“娘娘这次往哪逃？”
- 王妃摆好梯子：“不逃，计时。”
- 侍卫追来，反而被她安排成障碍物。

### 5～10 秒：宫斗变体育课

- 王妃翻过第三面墙，侍卫在后面打分；
- 皇帝看见全宫都在练翻墙，只能颁布“宫墙跑酷大赛”；
- 快切招生海报：**“三丈宫墙，七天结业。”**

## 终局状态

王妃不再争辩私逃案，改办宫墙跑酷营。本节点进入命运记录，不再继续生长。
