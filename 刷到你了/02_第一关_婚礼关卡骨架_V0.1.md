> 文档性质：首个原型关卡设计  
> 对应规则：《01_推荐流生成与视频增殖规则_V0.1.md》  
> 叙事节奏：《03_短视频叙事与时长规则_V0.1.md》

> 当前细化范围：关卡目标、玩家视角、逆天改命规则、第一阶段灯架事故、后续爽文主线

## 一、关卡定位

第一关是一场已经以死亡告终、需要玩家不断改写的荒诞婚礼。

初始婚礼视频从正常、简单的仪式开始，却在几秒内突然发生灯架坠落事故，新娘被砸死，视频至此无法继续。玩家不是预防一件尚未发生的事故，而是把物品送回这个死亡视频，反复改写命运，直到新娘活下来、婚礼能够进入下一阶段。

关卡体验由三部分组成：

- **逆天改命**：从已经无法继续的失败视频出发，把正确商品或服务送回关键时刻；
- **爽文**：玩家以新娘身份面对嘲笑和欺负，选择不同方式反击；
- **荒诞反馈**：商城可以买到的不只是物品，还可以是空调师傅等人物或服务。

## 二、玩家身份与目标

### 2.1 玩家身份

在婚礼关中，玩家代入新娘。

玩家不是控制新娘行走或直接选择台词，而是通过推荐流为新娘取得物品、人物和服务。结果视频负责展示新娘如何使用玩家送入的内容。

因此玩家的每次打赏同时具有两层含义：

- 玩法层：向某个视频投入解谜要素；
- 角色层：为“自己”取得解决问题或反击他人的手段。

### 2.2 明示目标

关卡开始时只显示一句结果要求：

> **让婚礼顺利结束。**

不显示目标视频封面，不公布需要经历多少阶段，也不直接告诉玩家哪件物品是正确答案。

### 2.3 第二层主线目标

除了明示结果，婚礼主线还包含一个不写在任务栏里的角色目标：

> **今天谁也别想让我受气。**

新娘被嘲笑、寻找证据、公开证据和完成反击都属于主线剧情。每个失败视频只有一个正确物品，必须按照固定顺序投入；香槟、礼花炮等其他配对进入荒诞终局。

婚礼主线中的每一个可改命视频都必须已经走到终点：死亡、婚礼取消，或新娘彻底离开婚礼。只有在原结果无法继续时，玩家才需要逆天改命。

## 三、视频与直播的统一生产规则

第一关的所有剧情反应都使用预先制作好的视频资产。

每条视频必须在前 5 秒讲清核心事件，首个原型单条视频不超过 15 秒。允许先展示死亡、取消婚礼或反击成功等结果，再回放导致结果的原因。

### 3.1 普通视频

玩家向普通视频打赏后：

1. 原视频暂停；
2. 界面播放通用礼物特效；
3. 原视频素材本身不改变；
4. 特效结束后立即播放一条新的预制结果视频；
5. 普通触发时原视频与结果视频同时保留；
6. 阶段结算时，已过时的前置视频退出推荐流。

### 3.2 直播间

直播间由预制视频与直播 UI 包装组成，不要求演员对玩家礼物进行实时表演。

玩家向直播间打赏后：

1. 直播素材暂停并播放通用礼物特效；
2. 特效结束后立即播放一条新的预制视频；
3. 新视频可以包装成直播切片、下播回应或主播刚发布的内容；
4. 直播间本身可以继续保留，除非阶段结算将其归档。

无论目标是普通视频还是直播间，每次有效打赏都统一产出 **1 条预制结果视频**。

## 四、可购买内容

第一关商城中的“礼物”分为三类：

- **物品**：梯子、香槟、礼花炮、录音笔等；
- **人物**：空调师傅、司仪、保安等；
- **服务**：紧急维修、现场投影、同城配送等。

人物和服务也以商品链接形式购买，并像礼物一样投入视频。荒诞感来自平台把一个完整的人或一项服务包装成可打赏商品。

购买人物不表示玩家拥有这个人，只表示获得一次“让该人物进入目标视频并提供服务”的投放机会。

## 五、初始推荐流

关卡初始只有 3 条内容。

### W001：婚礼灯架事故

**类型**：普通视频  
**账号**：婚礼跟拍账号  
**作用**：核心失败视频、第一阶段改命目标

视频内容：

- 视频开头是一场普通婚礼，仪式即将开始；
- 镜头偶尔拍到舞台上方灯架轻微晃动，但现场没人注意；
- 新娘走到舞台中央时灯架突然坠落；
- 新娘被灯架砸死，视频以黑屏、尖叫和新闻字幕收尾，不展示血腥画面；
- 婚礼无法继续，评论区开始追问吊顶和灯架是谁安装的；
- 画面线索表明灯架位置很高，需要梯子和懂吊顶设备的人处理。

### A001：这梯子什么都能够到

**类型**：商品视频或工具直播  
**挂链**：伸缩梯  
**作用**：提供第一件物品  

伸缩梯为免费教学商品或使用初始赠币购买，保证玩家开局即可取得。

### L001：同城空调维修直播间

**类型**：预制直播间  
**挂链**：空调师傅上门一次  
**作用**：提供第一位可购买人物  

直播内容以一本正经的方式售卖“空调师傅本人”，并强调“空调、吊顶、灯架附近的问题都能顺便看看”。购买后，背包中出现一次性礼物“空调师傅”。

## 六、第一阶段核心谜题

### 6.1 阶段问题

灯架固定点位于高处，同时现场没有懂吊顶设备的人。玩家需要把“梯子”和“空调师傅”都送回婚礼事故发生前。

这是一道双要素组合谜题：

`伸缩梯 + 空调师傅 + 死亡婚礼视频 = 灯架提前修好，新娘存活`

两件内容必须按固定顺序送入：先梯子，后空调师傅。

### 6.2 路线一：先送梯子

1. 玩家购买伸缩梯；
2. 把梯子打赏进 W001；
3. 立即播放 W101《伴郎爬上去了，但他拧错了》；
4. 视频中伴郎使用梯子检查灯架，却因为不懂结构拧错固定件，灯架依然坠落，新娘死亡；
5. W001 与 W101 暂时同时保留；
6. 玩家购买空调师傅；
7. 把空调师傅送进 W101；
8. 立即播放阶段结果 W200《师傅提前固定灯架，新娘活下来了》。

### 6.3 固定顺序规则

W001 的唯一正确物品是梯子，W101 的唯一正确物品是空调师傅。

如果玩家把空调师傅直接送进 W001，师傅会因为没有梯子无法登高，灯架依然坠落。该视频属于荒诞死亡终局，不生成 W200。玩家看完后返回 W001，仍需先送梯子。

因此第一阶段的正确路径固定为：

`W001 + 梯子 → W101`

`W101 + 空调师傅 → W200`

## 七、阶段结算

W200 播放完成后，新娘第一次摆脱死亡，第一阶段正式结算。

推荐流发生以下变化：

- W001 从当前推荐流退场；
- W101 从推荐流退场；
- A001 与 L001 作为已经完成教学职责的商品来源，从推荐流退场；
- 退场视频进入历史记录，可以回看但不能继续打赏；
- W200 留在推荐流，成为婚礼的当前状态；
- 梯子和空调师傅保持已发现状态；
- 玩家回退第一阶段时，W200 退出，相关前置视频恢复。

阶段结算后的推荐流数量会下降，形成清晰的“这一次命运终于被改写”反馈。

## 八、第二阶段主线：新娘反击

灯架事故是逆天改命教学。关卡下一阶段把矛盾从“让新娘活下来”转向“婚礼会不会因为羞辱与谎言而取消”，这一阶段仍然是必须推进的主线。

W200 之后出现主线失败视频 W300：刻薄伴娘当众嘲笑新娘寒酸，并播放经过剪辑的假证据污蔑新娘。宾客和新郎相信谎言，婚礼取消，新娘离场。视频再次无法沿婚礼线继续。

推荐流随后出现两条商品来源：

- **A002：录下她真正说了什么**——挂链商品为录音笔；
- **L002：婚礼大屏租赁直播**——挂链商品为投影服务。

第二阶段同样使用固定顺序：

1. W300 的唯一正确物品是录音笔；
2. 录音笔生成 W301《有完整录音，但现场没人相信》；
3. W301 仍以婚礼取消收尾，是新的可改命失败视频；
4. W301 的唯一正确物品是投影服务；
5. 投影服务生成 W400《完整证据公开，谎言被揭穿，婚礼继续》。

其他配对属于错误终局：

- **香槟**：爽文荒诞分支，新娘把酒泼回去，冲突爆红，但婚礼取消，新娘转入网红人生线；
- **礼花炮**：爽文荒诞分支，把伴娘炸成无伤害的爆炸头，现场变成网络综艺，婚礼不再继续；
- **投影服务直接送进 W300**：把伴娘剪辑过的假证据投得更大，婚礼更快取消；
- **梯子、空调师傅等不相关内容**：属于错误物品，投进嘲笑视频后只生成荒诞结果，不解决冲突。

正确路径固定为“先录音，再投影”。香槟、礼花炮和其他错误物品生成独立的终局视频：要么有人死亡，要么新娘进入另一条不再结婚的人生线。观看荒诞终局后，玩家回到对应失败节点继续寻找唯一正确物品。

## 九、分支控制

第一关追求高自由度，但所有分支都必须人工设计和制作视频，因此采用以下控制规则：

- 每个主线失败视频只配置一个正确物品；
- 正确物品投入正确视频后生成主线结果并推进状态；
- 其他配对生成荒诞终局：死亡或离开婚礼的另一条人生线；
- 生长节点可以继续接收部分礼物；
- 结果节点只负责播放、收藏、隐藏和回看，不再接受礼物；
- 荒诞分支默认是不可继续打赏的终局视频；
- 正确路径使用固定节点与固定投放顺序，不允许跳步或换序；
- 正确阶段结算会让已过时视频退场，防止推荐流无限膨胀；
- 错误投放也必须产生视频反馈，但不能替代正确解法；
- 关键物品误投后，玩家仍能通过再次购买或阶段回退取得主线所需内容。

## 十、首关制作预算

当前项目只制作和验证婚礼关。

首版建议把总量控制在以下范围：

- 初始内容：3 条；
- 主线生长与阶段结果：8～12 条；
- 婚礼主线改命阶段：8～12 条；
- 死亡或其他人生线的荒诞终局：6～10 条；
- 广告、商品演示和直播包装：4～6 条；
- 全关独立预制视频总量：约 27～35 条。

玩家单次通关预计看到 15～22 条内容，其余用于误投终局和重玩发现。

该预算是内容上限，不是必须填满的指标。原型首先验证 3 条初始内容、2 件按固定顺序投入的关键礼物和 1 次阶段结算是否有趣。

## 十一、本版已经锁定的规则

- 当前只设计和制作婚礼关；
- 玩家在这一关代入新娘；
- 关卡明示目标只有“让婚礼顺利结束”；
- 开局推荐流只有 3 条内容；
- 初始婚礼视频以新娘被灯架砸死收尾；
- 玩家只能对已经无法继续的失败视频进行逆天改命；
- 开局可以很快取得梯子；
- 另一条直播提供可购买的空调师傅；
- 人物与服务可以像物品一样被购买和打赏；
- 直播间与普通视频都使用预制素材；
- 每次有效打赏都立即自动播放 1 条预制结果视频；
- 原视频不即时生成演员反应；
- 普通触发让前后视频同时存在；
- 阶段解决后，已经过时的前置视频退出推荐流；
- 退场视频进入历史记录，回退阶段后才能重新互动；
- 第一阶段固定为“W001 塞梯子 → W101 塞空调师傅 → W200 新娘存活”；
- 第二阶段固定为“W300 塞录音笔 → W301 塞投影服务 → W400 婚礼继续”；
- 被嘲笑与完成爽文反击属于必须推进的主线阶段；
- 正确物品与正确视频配对才会推进主线，其他配对只生成荒诞分支；
- 荒诞分支必须以死亡或离开婚礼的另一条人生线收尾；
- 结果视频分为可继续生长与不可继续打赏两类。

## 十二、本版刻意不展开的内容

为了先验证核心自由度，本版不设计以下系统：

- 第二关及后续章节；
- 完整账号模拟经营；
- 完整乙游和男模好感系统；
- 跨关卡人格标签；
- 婚礼关后半段的全部事故链；
- 最终结局数量与完整视频树；
- 商业化、真实付费与联网功能。

后续讨论将继续围绕婚礼关逐阶段补全，不扩大到新的系统或关卡。
