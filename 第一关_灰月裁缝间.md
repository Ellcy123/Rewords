# 第一关：灰月裁缝间

## 关卡定位

**故事前提：** 一个不太富裕的女孩，想去参加今晚的贵族舞会。她没有钱、没有正式身份，也没有完整的邀请函。她拥有的只有阁楼裁缝间里那些被人看轻的旧物，以及把旧字重新炼成命运的能力。

**节点目标：** 在午夜前获得一种进入贵族舞会的身份或理由。

**节点调性：** 贫穷不是缺少光，而是光还没有被重新命名。

**核心数值：**
- 体面：影响贵族场合的接受度。
- 怀疑：影响门卫、贵族、继母阵营的盘问强度。
- 灵感：影响后续特殊合成与异常路线。
- 债务：影响女孩与旧账、赊欠、魔法代价之间的牵连。

---

## 一、分支剧情

### 分支 A：月裙入场

**触发结果卡：** 月裙

**结果状态：** 女孩以“被月光选中的陌生小姐”身份出现在舞会入口。她不显富贵，但显得不可解释。

**后续影响：**
- 开启“神秘小姐”路线。
- 后续贵族会更愿意谈论她，却没人能准确说出她来自哪一家。
- 王子相关节点更容易触发，但身份盘问也更容易出现。

**数值变化：**
- 体面 +1
- 灵感 +1
- 怀疑 +1

### 分支 B：铜帖通行

**触发结果卡：** 铜帖

**结果状态：** 女孩获得一份带有铜色印记的入场凭据。门口规则承认她，尽管人们未必承认她。

**后续影响：**
- 开启“凭证入场”路线。
- 后续可接触舞会名册、管家、门卫等秩序阵营。
- 若怀疑过高，铜帖会在后续节点被复查。

**数值变化：**
- 体面 +2
- 怀疑 -1
- 债务 +1

### 分支 C：镜门越界

**触发结果卡：** 镜门

**结果状态：** 女孩获得一种不经正门的入场资格。她不是被邀请进去的，而是被舞会的倒影接纳。

**后续影响：**
- 开启“镜中舞会”路线。
- 后续可能遇到镜中贵族、反向礼仪、倒影王子等异常事件。
- 普通贵族不易发现她的来处，但异常事件会增加。

**数值变化：**
- 灵感 +2
- 怀疑 +1
- 债务 +1

### 分支 D：旧账登门

**触发结果卡：** 旧账

**结果状态：** 女孩不以宾客身份出现，而以“被亏欠者”的身份站到贵族门前。舞会欠她一个解释。

**后续影响：**
- 开启“债与出身”路线。
- 可揭出女孩母亲、贵族家族、旧契约之间的关系。
- 后续更容易获得谈判资格，但也会招来继母或管家的压制。

**数值变化：**
- 体面 +1
- 怀疑 +2
- 债务 -1

## 二、结果卡牌

### 1. 月裙

**卡面图像：** 一条旧裙没有变成华服，只像把一整夜的月光穿在身上。裙摆仍旧朴素，但每一道折痕都亮得像秘密。

**合成配方：**
- 月（来源：月窗）
- 裙（来源：旧裙）

**对应分支：** 月裙入场

**数值效果：**
- 体面 +1
- 灵感 +1
- 怀疑 +1

**叙事短句：** 她没有宝石，只把夜色裁成了裙。

### 2. 铜帖

**卡面图像：** 一张薄帖压着暗铜色的光，像来自某个古老家族的迟到许可。

**合成配方：**
- 铜（来源：铜镜）
- 帖（来源：邀帖）

**对应分支：** 铜帖通行

**数值效果：**
- 体面 +2
- 怀疑 -1
- 债务 +1

**叙事短句：** 穷人没有请柬，于是她炼出一枚规则也认得的印。

### 3. 镜门

**卡面图像：** 铜镜里的房间多出一扇门，门后不是走廊，而是正在旋转的舞池灯影。

**合成配方：**
- 镜（来源：铜镜）
- 门（来源：门缝）

**对应分支：** 镜门越界

**数值效果：**
- 灵感 +2
- 怀疑 +1
- 债务 +1

**叙事短句：** 正门拒绝她，倒影却替她开门。

### 4. 旧账

**卡面图像：** 一页泛黄账纸浮出旧裙暗纹，欠下的名字像从灰里重新醒来。

**合成配方：**
- 旧（来源：旧裙）
- 账（来源：欠账）

**对应分支：** 旧账登门

**数值效果：**
- 体面 +1
- 怀疑 +2

## 三、双结果组合分支

玩家最多可合成 2 张结果卡。若合成 2 张，优先触发组合分支；组合分支继承两张卡的数值效果，并追加下列剧情方向。

### 月裙 + 铜帖：被邀请的月光

**结果状态：** 女孩既有入场凭据，又有足够令人记住的形象。她成为舞会中最安全、也最引人侧目的陌生人。

**追加影响：**
- 立即进入舞池外围。
- 王子节点触发率提高。
- 后续“身份复查”延后一个节点。

### 月裙 + 镜门：镜中月小姐

**结果状态：** 女孩以近乎传闻的方式出现。贵族们都记得她的裙光，却没人记得她从哪扇门进来。

**追加影响：**
- 开启“传闻”标签。
- 灵感再 +1。
- 后续异常事件权重提高。

### 月裙 + 旧账：穿着月光讨债

**结果状态：** 女孩不再像被亏欠的小人物，而像带着证词归来的继承人。

**追加影响：**
- 开启“失落继承人”标签。
- 管家阵营提前注意她。
- 后续可进入贵族档案节点。

### 铜帖 + 镜门：合法的幽灵

**结果状态：** 她拥有规则承认的凭据，却选择从规则背面进入。舞会记录上有她，现场没人看见她入场。

**追加影响：**
- 开启“无目击入场”标签。
- 怀疑暂时不增加，但后续一旦被追查，惩罚翻倍。

### 铜帖 + 旧账：债主的请柬

**结果状态：** 女孩的入场不再像伪装，而像一场迟来的清算。贵族家族必须接待她，至少在众目睽睽之下。

**追加影响：**
- 开启“公开谈判”标签。
- 债务再 -1。
- 后续敌对贵族发言更尖锐。

### 镜门 + 旧账：倒影里的欠债人

**结果状态：** 旧债被带进镜中，舞会的华丽倒影开始露出裂痕。

**追加影响：**
- 开启“诅咒账本”标签。
- 灵感 +1，债务 +1。
- 后续可进入黑童话路线。

## 四、场景物品

玩家进入关卡后获得以下全部场景物品卡，并全部拆字进入可用字池。

## 五、可用字池与配方表

**总字池：** 旧 / 裙 / 月 / 窗 / 铜 / 镜 / 门 / 缝 / 欠 / 账 / 邀 / 帖 / 煤 / 灰

**合法配方仅限：**
- 月 + 裙 → 月裙
- 铜 + 帖 → 铜帖
- 镜 + 门 → 镜门
- 旧 + 账 → 旧账

不在上述表中的组合无法合成结果卡。

## 六、剩余字处理示例

基础组句词：**我 / 你 / 在 / 了**

### 示例 1：只合成“月裙”

**消耗字：** 月 / 裙

**剩余字：** 旧 / 窗 / 铜 / 镜 / 门 / 缝 / 欠 / 账 / 邀 / 帖 / 煤 / 灰

**可尝试句：**
- 我在旧窗了，铜镜门缝欠账邀帖煤灰。
- 你在门缝邀帖了，我欠旧账，煤灰在铜镜。

**预期反馈：**
- 合理分低，抽象分高。
- 因剩余字多，怀疑或债务可能上升。
- 适合教学：只炼一张卡很容易成功，但代价会留在句子里。

### 示例 2：合成“月裙 + 铜帖”

**消耗字：** 月 / 裙 / 铜 / 帖

**剩余字：** 旧 / 窗 / 镜 / 门 / 缝 / 欠 / 账 / 邀 / 煤 / 灰

**可尝试句：**
- 我在旧窗镜门了，你欠账邀煤灰。
- 旧窗在我，镜门在你，欠账邀煤灰了。

**预期反馈：**
- 合理分中低，诗意分中。
- “旧窗、镜门、煤灰”会形成强意象，适合 AI 给出有趣反馈。

### 示例 3：合成“铜帖 + 旧账”

**消耗字：** 铜 / 帖 / 旧 / 账

**剩余字：** 裙 / 月 / 窗 / 镜 / 门 / 缝 / 欠 / 邀 / 煤 / 灰

**可尝试句：**
- 月在裙窗了，镜门缝欠邀煤灰。
- 我在月窗，裙镜门缝欠邀煤灰了。

**预期反馈：**
- 合理分低，诗意分高。
- 因“欠”被剩下，AI 可追加负担：“你把欠留在了身后，但它没有留在房间里。”

## 七、设计自检

- [x] 从剧情分支开始设计，而不是从场景物品开始。
- [x] 结果卡牌恰好 4 张。
- [x] 每张结果卡牌都指向独立剧情分支。
- [x] 不存在辅助卡或工具卡。
- [x] 场景物品自然存在于贫穷女孩的阁楼裁缝间。
- [x] 玩家拿到全部场景卡后，可合成多张结果卡。
- [x] 存在多种 2 张结果卡组合，并配置组合分支。
- [x] 不同合成选择会产生不同剩余字数量、意象和负担。
- [x] 非配方表中的组合不能合成结果卡。
