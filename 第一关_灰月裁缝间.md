# 第一关：灰月裁缝间

## 关卡定位

**故事前提：** 一个不太富裕的女孩想去参加今晚的贵族舞会。她没有礼服，也没有正式身份。阁楼裁缝间里只有旧物、残片和一点点不肯熄灭的念头。

**节点目标：** 在出门前炼出一种能去舞会的身份、凭证或亮点。

**设计重点：** 第一关只做“少字、好拼、强啊哈”。玩家不需要面对很大的字池，但要能明显感到：普通旧物里的字，拼出了童话关键词。

**核心数值：**
- 体面：决定贵族场合是否接纳她。
- 怀疑：决定她会不会被盘问、揭穿。
- 灵感：决定她能否触发童话式事件。

---

## 一、分支剧情

### 分支 A：童话身份

**触发结果卡：** 灰姑娘

**结果状态：** 女孩被重新命名为“灰姑娘”。贫穷不再只是羞耻，也变成一个会被记住的故事开头。

**后续影响：**
- 开启“童话主角”路线。
- 后续更容易触发王子、午夜、遗失物事件。

**数值变化：**
- 灵感 +2
- 体面 +1
- 怀疑 +1

### 分支 B：舞会亮点

**触发结果卡：** 水晶鞋

**结果状态：** 女孩得到一件足够吸引视线的奇迹物证。她还不一定像贵族，但已经不像普通来客。

**后续影响：**
- 开启“遗失之鞋”路线。
- 后续王子或侍从更容易注意到她。

**数值变化：**
- 体面 +2
- 灵感 +1

### 分支 C：凭证入场

**触发结果卡：** 请柬

**结果状态：** 女孩获得一份能被门卫理解的入场理由。它不华丽，但足够让规则放她进去。

**后续影响：**
- 开启“正门入场”路线。
- 后续盘问较少，但不一定能进入贵族核心圈。

**数值变化：**
- 体面 +1
- 怀疑 -1

### 分支 D：伪装身份

**触发结果卡：** 贵族姑娘

**结果状态：** 女孩以“贵族姑娘”的身份出现。这个身份听起来体面，但它太体面了，反而容易被人追问出身。

**后续影响：**
- 开启“伪装贵族”路线。
- 可更快接近贵族圈层。
- 后续若怀疑过高，会触发“你是哪一家的小姐？”盘问。

**数值变化：**
- 体面 +3
- 怀疑 +2

## 二、结果卡牌

### 1. 灰姑娘

**卡面图像：** 煤灰从裙角浮起，和旧画像里的“姑娘”拼成一个新名字。

**合成配方：**
- 灰（来源：煤灰裙）
- 姑（来源：姑娘像）
- 娘（来源：姑娘像）

**对应分支：** 童话身份

**数值效果：**
- 灵感 +2
- 体面 +1
- 怀疑 +1

**啊哈点：** “煤灰”里的灰，加上“姑娘像”里的姑娘，直接拼出“灰姑娘”。玩家会觉得自己不是做道具，而是在拼出主角。

### 2. 水晶鞋

**卡面图像：** 水晶灯的一点冷光落进旧舞鞋，鞋面亮得像透明的冰。

**合成配方：**
- 水（来源：水晶灯）
- 晶（来源：水晶灯）
- 鞋（来源：旧舞鞋）

**对应分支：** 舞会亮点

**数值效果：**
- 体面 +2
- 灵感 +1

**啊哈点：** “水晶”原本是灯，“鞋”原本是旧鞋，拼在一起就变成童话里最有记忆点的物品。

### 3. 请柬

**卡面图像：** 撕坏的请柬角被重新拼齐，烫金边缘重新亮起。

**合成配方：**
- 请（来源：请柬角）
- 柬（来源：请柬角）

**对应分支：** 凭证入场

**数值效果：**
- 体面 +1
- 怀疑 -1

**啊哈点：** 玩家从“请柬角”里提炼出完整“请柬”，这是最直观、最安全的新手合成。

### 4. 贵族姑娘

**卡面图像：** 报纸上的贵族剪影和旧画像中的姑娘重叠，像一张临时编出来的身份。

**合成配方：**
- 贵（来源：贵族报）
- 族（来源：贵族报）
- 姑（来源：姑娘像）
- 娘（来源：姑娘像）

**对应分支：** 伪装身份

**数值效果：**
- 体面 +3
- 怀疑 +2

**啊哈点：** “贵族”和“姑娘”都是现成词，合在一起变成一个能骗人、也会惹麻烦的身份。

## 三、双结果组合分支

玩家最多可合成 2 张结果卡。若合成 2 张，优先触发组合分支。

### 灰姑娘 + 水晶鞋：童话开场

**结果状态：** 她既有童话身份，又有童话物证。舞会会把她当成今晚的传闻中心。

**追加影响：**
- 王子路线强触发。
- 后续必定出现“遗失之鞋”事件。

### 灰姑娘 + 请柬：被邀请的灰姑娘

**结果状态：** 她的故事听起来离奇，但手里有凭证。门卫愿意放行，贵族愿意多看一眼。

**追加影响：**
- 正门入场。
- 首次身份盘问时怀疑 -1。

### 水晶鞋 + 请柬：低风险入场

**结果状态：** 她有请柬，也有亮点。这个路线不最戏剧化，但最稳。

**追加影响：**
- 体面额外 +1。
- 后续进入舞池外围。

### 贵族姑娘 + 水晶鞋：假身份，真奇迹

**结果状态：** 她的身份可能是假的，但鞋是真的。众人想怀疑，却又被奇迹吸引。

**追加影响：**
- 可接近贵族核心圈。
- 后续揭穿风险提高。

### 贵族姑娘 + 请柬：像真的小姐

**结果状态：** 她看起来有身份，也拿得出凭证。短时间内，没人愿意当众拆穿她。

**追加影响：**
- 当前节点怀疑不再增加。
- 后续触发家族盘问事件。

## 四、场景物品

玩家进入关卡后获得以下全部场景物品卡，并全部拆字进入可用字池。

### 1. 煤灰裙

**图像：** 一条沾着煤灰的旧裙，裙角洗不出原来的颜色。

**可拆字：** 煤 / 灰 / 裙

**类型：** 物品

**用途：**
- 灰 → 灰姑娘
- 煤 / 裙 → 剩余字

### 2. 姑娘像

**图像：** 墙上挂着一幅褪色少女画像，像在看着她准备出门。

**可拆字：** 姑 / 娘 / 像

**类型：** 物品

**用途：**
- 姑 / 娘 → 灰姑娘、贵族姑娘
- 像 → 剩余字

### 3. 水晶灯

**图像：** 一盏坏掉的小水晶灯，还剩几片会反光的坠子。

**可拆字：** 水 / 晶 / 灯

**类型：** 物品

**用途：**
- 水 / 晶 → 水晶鞋
- 灯 → 剩余字

### 4. 旧舞鞋

**图像：** 一只旧舞鞋，鞋尖磨破，却还保留着跳舞的形状。

**可拆字：** 旧 / 舞 / 鞋

**类型：** 物品

**用途：**
- 鞋 → 水晶鞋
- 旧 / 舞 → 剩余字

### 5. 请柬角

**图像：** 垃圾篓里露出的半片请柬，缺了姓名和日期。

**可拆字：** 请 / 柬 / 角

**类型：** 物品

**用途：**
- 请 / 柬 → 请柬
- 角 → 剩余字

### 6. 贵族报

**图像：** 一张旧报纸，报道今晚贵族舞会和来宾名单。

**可拆字：** 贵 / 族 / 报

**类型：** 物品

**用途：**
- 贵 / 族 → 贵族姑娘
- 报 → 剩余字

## 五、可用字池与配方表

**总字池：** 煤 / 灰 / 裙 / 姑 / 娘 / 像 / 水 / 晶 / 灯 / 旧 / 舞 / 鞋 / 请 / 柬 / 角 / 贵 / 族 / 报

**合法配方仅限：**
- 灰 + 姑 + 娘 → 灰姑娘
- 水 + 晶 + 鞋 → 水晶鞋
- 请 + 柬 → 请柬
- 贵 + 族 + 姑 + 娘 → 贵族姑娘

不在上述表中的组合无法合成结果卡。

## 六、剩余字处理示例

基础组句词：**我 / 你 / 在 / 了**

### 示例 1：合成“请柬”

**消耗字：** 请 / 柬

**剩余字：** 煤 / 灰 / 裙 / 姑 / 娘 / 像 / 水 / 晶 / 灯 / 旧 / 舞 / 鞋 / 角 / 贵 / 族 / 报

**可尝试句：**
- 我在煤灰裙了，你在姑娘像。
- 水晶灯旧舞鞋在角，贵族报在我。

**预期反馈：**
- 最稳的结果卡，但剩余字很多。
- 玩家会理解：只合成安全凭证，生活里的杂物仍要处理。

### 示例 2：合成“灰姑娘 + 水晶鞋”

**消耗字：** 灰 / 姑 / 娘 / 水 / 晶 / 鞋

**剩余字：** 煤 / 裙 / 像 / 灯 / 旧 / 舞 / 请 / 柬 / 角 / 贵 / 族 / 报

**可尝试句：**
- 我在煤裙像灯了。
- 旧舞请柬角在贵族报。

**预期反馈：**
- 合理分中低，诗意分较高。
- 这是一条最有童话感的组合，但也留下“贵族报”等身份隐患。

### 示例 3：合成“贵族姑娘 + 请柬”

**消耗字：** 贵 / 族 / 姑 / 娘 / 请 / 柬

**剩余字：** 煤 / 灰 / 裙 / 像 / 水 / 晶 / 灯 / 旧 / 舞 / 鞋 / 角 / 报

**可尝试句：**
- 我在煤灰裙像了。
- 水晶灯旧舞鞋角报在你。

**预期反馈：**
- 体面高，但“煤灰裙”留在剩余字里，容易被 AI 评为身份负担。
- 教学意义：越想伪装，越要处理被留下的出身痕迹。

## 七、设计自检

- [x] 从剧情分支开始设计。
- [x] 结果卡牌恰好 4 张。
- [x] 每张结果卡牌都指向独立剧情分支。
- [x] 结果卡不是工具卡，而是身份、物证、凭证、伪装四种叙事结果。
- [x] 结果词具有“啊哈感”：灰姑娘、水晶鞋、请柬、贵族姑娘。
- [x] 场景物品控制为 6 张，总字池 18 个，适合第一关教学。
- [x] 玩家拿到全部场景卡后，可合成多张结果卡，但规则限制只能选 1-2 张。
- [x] 存在多种 2 张结果卡组合，并配置组合分支。
- [x] 非配方表中的组合不能合成结果卡。
